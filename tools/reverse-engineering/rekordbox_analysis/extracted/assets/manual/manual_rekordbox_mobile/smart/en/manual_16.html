Organizing Music in Playlists

Playlists can be created according to your DJ performance requirements to organize the library.

1When [Prepare] is tapped at the top menu, the browser panel opens.

2Tap the 【PLAYLIST】 category.

A list of playlists and folders is displayed.

3When is tapped, the playlist name input box opens.

4Input the playlist name, then tap [OK].

The newly created playlist is added to the 【PLAYLIST】 category.

5Tap the newly created playlist to open it.

6Tap [Edit] and then tap .

A panel for selecting tracks to be added opens.

7Tap the tracks you want to add to the playlist. (Multiple tracks can be selected.)

8Tap [Done] to finish adding tracks.

9Tap [Done] to finish playlist editing.

Editing existing playlists

1Tap the 【PLAYLIST】 category.

A list of playlists and folders is displayed.

2Tap the playlist which you want to edit to open it.

3Tap [Edit], then add or delete tracks and change the track order.

Icon

Description

When tapped, a panel for selecting tracks to be added to the playlists opens.

When dragged, the order of the tracks can be changed.

When tapped, [Remove] is displayed. Tap [Remove] to delete tracks.

4Tap [Done] to finish playlist editing.

Organizing playlists

1Tap the 【PLAYLIST】 category.

A list of playlists and folders is displayed.

2When is tapped, the folder name input box opens.

3Input the folder name, then tap [OK].

The created folder is added to the 【PLAYLIST】 category.

4Tap [Edit], then move or delete the playlist or folder.

Icon

Description

Use this to create a new playlist.

Use this to create a new folder.

Playlists and folders can be moved by dragging them. To move a playlist or folder into a folder in a lower level, drag it over the folder. The level below is displayed. To move a playlist or folder into a folder in a higher level, drag it to the left edge of the screen. The level above is displayed.

When the playlist name is touched and held, the playlist name input box opens. When the playlist name is input then [OK] is tapped, the playlist name is edited.

When the folder name is touched and held, the folder name input box opens. When the folder name is input then [OK] is tapped, the folder name is edited.

When this is tapped, [Delete] is displayed. Tap [Delete] to delete the selected playlist or folder.

  • When a folder is deleted, all the playlists and folders in that folder are deleted.

Enlarge table

5Tap [Done] to finish editing the playlists and folders.

Verknüpfung mit einer XDJ-Einheit

Zutreffende Modelle: XDJ-AERO

Mobile Geräte können über WLAN an Pioneer DJ-Einheiten, die rekordbox (iOS/Android) unterstützen, zum Austausch von Musikdaten angeschlossen werden.

1Führen Sie den Anschluss an das Netzwerk mit der internen WLAN-Funktion der DJ-Einheit aus.

  • Einzelheiten über WLAN-Verbindungstypen und Verbindungsverfahren siehe Bedienungsanleitung der betreffenden DJ-Einheit.

  • XDJ-AERO-Anschlussanleitung

2Verbinden Sie das mobile Gerät, auf dem rekordbox (iOS/Android) installiert ist, mit dem WLAN-Netzwerk.

  • Für Anweisungen zum Herstellen der WLAN-Verbindungen siehe Bedienungsanleitung Ihres mobilen Geräts.

3Im rekordbox (iOS/Android) Top-Menü tippen Sie auf [Load].

Wenn eine DJ-Einheit, mit der Kommunikation möglich ist, gefunden wird, versucht rekordbox (iOS/Android) automatisch, die Verbindung herzustellen.

Wenn die Verknüpfung hergestellt ist, wird das Browser-Feld geöffnet.

Hinweise

Bis zu vier mobile Geräte können gleichzeitig mit der DJ-Einheit verknüpft werden.

Der Name des mobilen Geräts, das an der DJ-Einheit angezeigt wird, kann mit rekordbox (iOS/Android) geprüft und geändert werden.

  • Voreinstellungen

Android-Geräte, die den MSC-Modus unterstützen, arbeiten als USB-Speichergeräte (sekundäre Speichergeräte wie USB-Flash-Speicher-Laufwerke), wenn sie an die DJ-Einheit über ein USB-Kabel angeschlossen werden. In diesem Fall können mit rekordbox (iOS/Android) keine Daten übertragen werden. Einzelheiten zur Verbindung zwischen DJ-Einheiten und USB-Speichergeräten siehe Bedienungsanleitung der betreffenden DJ-Einheit.

Abbrechen der Verknüpfung mit der DJ-Einheit

Wenn Sie auf dem Browser-Feld auf oder auf [Menu] auf der Symbolleiste tippen und dann zum Top-Menü zurückgehen, wird die Verknüpfung zur DJ-Einheit abgebrochen.

用語解説

BEAT GRID (ビートグリッド)

音楽ファイルを解析して検出された拍 （ビート） が、 拡大波形表示上に点や線でグリッド表示されます。

CUE/LOOP POINT MEMORY (キュー/ループポイントメモリー)

あらかじめキュー/ループポイントを設定・保存しておくことによって、 DJプレーヤーでキュー/ループポイントを呼び出せます。 rekordboxで保存できるキュー/ループポイントは、 1つの音楽ファイルにつき10個までです。

HISTORY (演奏履歴)

DJプレーヤーで演奏した演奏履歴をrekordboxにフィードバックできます。 演奏履歴はrekordboxにプレイリストとして保存できます。

HOT CUE (ホットキュー)

キューポイントやループポイントをホットキューに記録しておくことによって、 DJプレーヤーでワンプッシュで頭出し再生やループ再生がスタートできます。 rekordboxで記録できるホットキューは、 1つの音楽ファイルにつき3個までです。 (※XDJ-RXは最大4個のホットキューを記録できます。)

LINK EXPORT

DJプレーヤーとコンピューター (またはモバイルデバイス) をLANケーブルや無線LAN、 またはUSBケーブルで接続すると、 DJプレーヤーにrekordboxの音楽ファイルやデータをリアルタイムでロードできます。

PLAYLIST (プレイリスト)

DJプレイの用途に合わせたプレイリストを作って、 rekordboxの音楽コレクションを整理できます。 お気に入りの音楽ファイルのプレイリストをあらかじめ作成しておくことによって、 DJプレーヤーで目的の音楽ファイルを素早く選曲できます。

PRO DJ LINK

PRO DJ LINKに対応したPioneer DJ製DJ機器をLANケーブルを使って相互に接続することによって、 DJ ミキサーや複数のDJプレーヤーでrekordboxの音楽ファイルやデータを共有できます。 ネットワークに参加できるDJプレーヤーは4台までです。 (※XDJ-AERO/XDJ-RXは1台だけネットワークに参加ができます。XDJ-AERO/XDJ-RXは他のDJ機器と同じネットワークに参加できません。) また、 ネットワークに無線LANのアクセスポイントがあれば、 本ソフトウェアがインストールされているモバイルデバイスも4台までネットワークに参加できるので、 スムーズにDJが入れ替わることが可能になります。

QUANTIZE (クオンタイズ)

キューやループやホットキューが拍 （ビート） の位置に合わせて設定されます。 rekordboxを使って音楽ファイルの拍 （ビート） の位置を検出し、 ビートグリッドをあらかじめ調整しておくことによって、 DJ機器でラフにボタンを押しても、 オンビートでキューやループ、 エフェクトを設定・演奏できます。

TAG LIST (タグリスト)

時々のDJ シーンに応じて選んだ曲にマークを付けてタグリストに入れることができます。 タグリストはLANで接続された複数のDJプレーヤーとの間でオンラインで共用できます。 各DJプレーヤーからの操作で、 タグリスト上の音楽ファイルをそれぞれのDJプレーヤーにロードして再生したり、 実際の演奏に使われたタグリストをrekordboxのプレイリストとして保存したりできます。

  • DJプレーヤーからタグリストへのアクセス方法については、 各DJプレーヤーの取扱説明書をご覧ください。

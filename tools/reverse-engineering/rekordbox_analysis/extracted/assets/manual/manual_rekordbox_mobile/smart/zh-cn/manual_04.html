rekordbox面板

顶部菜单

启动rekordbox（iOS/Android）时，将出现顶部菜单。

Prepare

用于创建不同类型的点信息、 播放列表等以事先为DJ表演做好准备。

  • 浏览器面板

  • 播放机面板

Load

用于链接到DJ播放机以用于DJ表演。

  • 链接到CDJ/XDJ设备

  • 链接到XDJ设备

Connect

用于将已利用计算机上的rekordbox（Mac/Windows）上分析的音乐文件、 不同类型的点信息、 播放列表等导入rekordbox（iOS/Android）。 也用于反映rekordbox（Mac/Windows）中利用rekordbox（iOS/Android）更新的不同类型的点信息、 播放列表等。

  • 从计算机导入

  • 将更新信息发送到计算机

Import

用于对移动设备上的音乐文件进行分析并将它们添加到rekordbox（iOS/Android）库。

  • 使用移动设备上的音乐文件

Help

打开帮助菜单。

Settings

用于检查并更改不同设定。

  • 首选项

工具栏

有关屏幕底部工具栏上显示的图标，请参阅下表。

图标

说明

Menu

显示顶部菜单。

Browse

返回到浏览器面板的最上层。

Playing

用于打开播放机面板并显示当前播放的曲目。

USB

用于浏览连接到DJ设备（XDJ-AERO）的USB设备上的内容并在DJ设备（XDJ-AERO）上播放。

  • 有关操作USB设备的说明，请参阅相应的DJ设备使用说明书。

Import

用于对移动设备上的音乐文件进行分析并将它们添加到rekordbox（iOS/Android）库。

  • 使用移动设备上的音乐文件

Settings

用于检查并更改不同设定。

  • 首选项

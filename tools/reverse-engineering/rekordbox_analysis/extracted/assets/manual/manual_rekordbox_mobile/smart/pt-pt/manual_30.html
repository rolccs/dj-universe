Ambiente de rede

Dependendo das definições do software de segurança, o sistema operativo, etc., a ser utilizado no computador, as comunicações entre o computador e o dispositivo móvel pode ser interrompido.

  • Para obter informações sobre o ambiente de comunicações necessário no computador (programas, sistema operativo, rede, etc.), consulte as instruções de funcionamento do rekordbox (Mac/Windows).

Além disso, se a rede (endereços IP, números de porta, etc.) for restringida pelo dispositivo de comunicações (router, etc.), a comunicação com computadores ou unidades de DJ pode ser interrompida.

  • Para obter instruções sobre como configurar os dispositivos de comunicação que está a utilizar, contacte os respectivos fabricantes ou representantes de vendas.

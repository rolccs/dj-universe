Browser-Feld

Dient zum Zurückkehren zur darüberliegenden Ebene.

Die momentan durchsuchte Lage (Ebene) wird hier angezeigt.

  • Durchsuchen von Musikdateien

Bei Verknüpfung mit DJ-Playern wird der Wiedergabestatus der DJ-Player angezeigt.

Eine Liste der Alben, Titel usw. in der momentan durchsuchten Lage (Ebene) wird hier angezeigt. Wenn ein Track angetippt wird, wird das Player-Feld geöffnet, und die Wiedergabe beginnt.

Wenn ein Track Verknüpfung mit DJ-Playern angetippt wird, wird das Verknüpfungen-Feld (CDJs/XDJs) oder das Verknüpfungen-Feld (XDJ) geöffnet.

Dient zum Scrollen der Liste zu der mit dem angegebenen Buchstaben beginnenden Zeile.

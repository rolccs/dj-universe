Collegamento con un computer

Il dispositivo mobile può essere collegato via LAN wireless ad un computer su cui sia installato rekordbox (Mac/Windows) per trasferire dati musicali fra di esso e rekordbox (Mac/Windows).

A seconda delle impostazioni dei software di sicurezza, del sistema operativo, ecc. usati nel computer, la comunicazione fra il computer ed il dispositivo mobile può interrompersi. Per dettagli sull’ambiente di comunicazione richiesto dal computer (programmi, sistema operativo, rete, ecc.), vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

1Collegare il computer su cui è installato rekordbox (Mac/Windows) alla rete LAN wireless.

  • Per istruzioni sui collegamenti via LAN wireless, vedere le istruzioni per l’uso per l’uso del computer o dispositivo per LAN wireless.

2Collegare il dispositivo mobile su cui è installato rekordbox (iOS/Android) alla rete LAN wireless.

  • Per istruzioni sui collegamenti via LAN wireless, vedere le istruzioni per l’uso del dispositivo mobile e dispositivo per LAN wireless.

3Lanciare rekordbox (Mac/Windows) nel computer.

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

4Nel menu principale di rekordbox (iOS/Android), toccare [Connect].

rekordbox (iOS/Android) rimane in attesa dell’operazioni di collegamento (ricerca di dispositivi mobili) to be performed con rekordbox (Mac/Windows) nel computer.

5Aprire il riquadro [Dispositivo] in rekordbox (Mac/Windows) del computer e fare clic su in basso a sinistra nel riquadro [Dispositivo].

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

Stabilito il collegamento, viene visualizzato il nome del dispositivo mobile nel riquadro [Dispositivo] di rekordbox (Mac/Windows).

Note

Si può collegare a rekordbox (Mac/Windows) nel computer solo un dispositivo mobile alla volta.

Il nome del dispositivo mobile visualizzato nel riquadro [Dispositivo] è controllabile e modificabile con rekordbox (iOS/Android).

Collegamento ad un computer via cavo USB

I dispositivi Android che supportano la modalità MSC funzionano come dispositivi di memoria USB (dispositivi di memoria secondari, ad esempio unità di memoria flash USB) se collegati ad un computer via cavo USB.

Collegare il dispositivo Android impostato sulla modalità MSC al computer usando un cavo USB, quindi lanciare rekordbox (Mac/Windows). Il nome del dispositivo mobile (dispositivo Android) viene visualizzato nel riquadro [Dispositivo] in rekordbox (Mac/Windows).

  • Per sapere come impostare la modalità MSC, leggere le istruzioni per l’uso del dispositivo Android. Se il computer non riconosce il vostro dispositivo Android come dispositivo di memoria USB (dispositivo di memoria secondario, ad esempio un’unità di memoria flash USB), consultare il fabbricante del dispositivo Android o il negozio di acquisto.

Cancellazione del collegamento col computer

Per cancellare il collegamento, aprire il riquadro [Dispositivo] in rekordbox (Mac/Windows) del computer e fare clic su in basso a destra dell’icona del dispositivo.

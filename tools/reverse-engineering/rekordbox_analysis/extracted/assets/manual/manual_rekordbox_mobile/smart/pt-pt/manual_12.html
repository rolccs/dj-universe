Importar a partir de um computador

As listas de reprodução e ficheiros de música analisados com o rekordbox (Mac/Windows) no computador podem ser importados para o rekordbox (iOS/Android).

Antes de tentar importar ficheiros de música, listas de reprodução, etc., a partir do rekordbox (Mac/Windows) no computador, estabeleça a ligação entre o dispositivo móvel e o computador.

  • Ligar a um computador

1Abra os painéis [Dispositivo] e [Colecção] próximos entre si no rekordbox (Mac/Windows) no computador.

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

2Arraste o ficheiro de música do painel [Colecção] para o painel [Dispositivo].

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

  • O nome do dispositivo móvel exibido no painel [Dispositivo] pode ser verificado e alterado utilizando o rekordbox (iOS/Android).

    • Preferências

Os vários tipos de informações de point, etc., preparados com rekordbox (Mac/Windows) são importados para a biblioteca do rekordbox (iOS/Android) juntamente com os ficheiros de música.

Nota

Para cancelar a ligação com o computador, clique em à direita do ícone do dispositivo no painel [Dispositivo] no rekordbox (Mac/Windows).

Importar listas de reprodução

As listas de reprodução criadas com o rekordbox (Mac/Windows) no computador podem ser importadas para o rekordbox (iOS/Android).

1Abra os painéis [Dispositivo] e [Lista de Reprodução] próximos entre si no rekordbox (Mac/Windows) no computador.

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

2Arraste a lista de reprodução do painel [Lista de Reprodução] para o painel [Dispositivo].

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

Os vários tipos de informações de point, etc., preparados com rekordbox (Mac/Windows) são importados para a biblioteca do rekordbox (iOS/Android) juntamente com as listas de reprodução e ficheiros de música.

Importar valores de “Minhas definições”

Também é possível importar os valores de “Minhas definições” armazenados no rekordbox (Mac/Windows) no computador para rekordbox (iOS/Android).

1No rekordbox (Mac/Windows) no computador, seleccione [Preferências] no menu [Ficheiro] para abrir o ecrã [Preferências].

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

2No ecrã [Preferências], seleccione [Exportando] > [Minhas definições] e, em seguida, abra o separador [Minhas definições].

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

3No separador [Minhas definições], clique em [Aplicar a todos os dispositivos ligados].

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

  • Quando [Aplicar automaticamente as alterações aos dispositivos] é seleccionado, os valores de “Minhas definições” preparados com o rekordbox (Mac/Windows) são importados automaticamente para o rekordbox (iOS/Android) a partir da próxima vez.

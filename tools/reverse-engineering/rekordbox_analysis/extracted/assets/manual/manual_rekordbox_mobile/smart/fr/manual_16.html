Organisation de musique dans des listes de lecture

Pour organiser votre bibliothèque selon le type de prestation DJ que vous envisagez, des listes de lecture peuvent être créées.

1Lorsque [Prepare] est tapé dans le menu principal, le panneau d’exploration s’ouvre.

2Tapez sur la catégorie 【PLAYLIST】.

Une liste des listes de lecture et de dossiers apparaît.

3Lorsque est tapé, la case de saisie du nom de liste de lecture s’ouvre.

4Saisissez le nom de la liste de lecture, puis tapez sur [OK].

La liste de lecture nouvellement créée est ajoutée à la catégorie 【PLAYLIST】.

5Tapez sur la liste de lecture nouvellement créée pour l’ouvrir.

6Tapez sur [Modifier] puis sur .

Le panneau de sélection des morceaux à ajouter s’ouvre.

7Tapez sur les morceaux que vous voulez ajouter à la liste de lecture. (Plusieurs morceaux peuvent être sélectionnés.)

8Tapez sur [OK] pour valider l’ajout de morceaux.

9Tapez sur [OK] pour valider l’édition de la liste de lecture.

Édition de listes de lecture existantes

1Tapez sur la catégorie 【PLAYLIST】.

Une liste des listes de lecture et de dossiers apparaît.

2Tapez sur la liste de lecture que vous voulez éditer pour l’ouvrir.

3Tapez sur [Modifier], puis ajoutez ou supprimez des morceaux et changez l’ordre des morceaux.

Icône

Description

Lorsque vous tapez dessus, le panneau de sélection des morceaux à ajouter aux listes de lecture s’ouvre.

L’ordre des morceaux peut être changé par un glisser.

Lorsque vous tapez dessus, [Retirer] apparaît. Tapez sur [Retirer] pour supprimer des morceaux.

Agrandir le tableau

4Tapez sur [OK] pour valider l’édition de la liste de lecture.

Organisation de listes de lecture

1Tapez sur la catégorie 【PLAYLIST】.

Une liste des listes de lecture et de dossiers apparaît.

2Lorsque est tapé, la case de saisie du nom de dossier s’ouvre.

3Saisissez le nom du dossier, puis tapez sur [OK].

La dossier créé est ajouté à la catégorie 【PLAYLIST】.

4Tapez sur [Modifier] puis déplacez ou supprimez la liste de lecture ou le dossier.

Icône

Description

Utilisez ceci pour créer une nouvelle liste de lecture.

Utilisez ceci pour créer un nouveau dossier.

Les listes de lecture et les dossiers peuvent être déplacés par un glisser. Pour mettre une liste de lecture ou un dossier dans un dossier d’un niveau inférieur, faites-le glisser par dessus le dossier. Le niveau inférieur apparaît. Pour mettre une liste de lecture ou un dossier dans un dossier d’un niveau supérieur, faites-le glisser vers la gauche de l’écran. Le niveau supérieur apparaît.

Lorsque vous touchez le nom d’une liste de lecture et maintenez le doigt dessus, la case de saisie du nom de liste de lecture s’ouvre. Lorsque vous saisissez le nom de la liste de lecture puis tapez sur [OK], le nom de liste de lecture change.

Lorsque vous touchez le nom d’un dossier et maintenez le doigt dessus, la case de saisie du nom dossier s’ouvre. Lorsque vous saisissez le nom du dossier puis tapez sur [OK], le nom du dossier change.

Lorsque vous tapez sur ceci, [Supprimer] apparaît. Tapez sur [Supprimer] pour supprimer la liste de lecture ou le dossier sélectionné.

  • Quand un dossier est supprimé, toutes les listes de lecture et tous les dossiers dans ce dossier sont supprimés.

5Tapez sur [OK] pour valider l’édition des listes de lecture et des dossiers.

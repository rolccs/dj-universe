Seleccionar música para uma lista de tags

A lista de tags é uma lista que pode ser partilhada em tempo real entre vários leitores de DJ ligados numa LAN.

1Quando prime continuamente uma faixa no painel do browser, o ecrã de acção é aberto.

2Quando toca em [Adicionar à Lista de Tags], a faixa seleccionada é adicionada à categoria 【TAGLIST】.

  • Para obter instruções sobre como aceder às listas de tags a partir dos leitores de DJ, consulte as instruções de funcionamento dos respectivos leitores de DJ.

Eliminar faixas de listas de tags

Para eliminar uma faixa de cada vez

1No painel do browser, toque na categoria 【TAGLIST】.

É exibida uma lista com as faixas que foram adicionadas à lista de tags.

2Quando prime continuamente a faixa que pretende eliminar, o ecrã de acção é aberto.

3Quando toca em [Remover da Lista de Tags], a faixa seleccionada é eliminada da categoria 【TAGLIST】.

Para eliminar todas as faixas

1Quando prime continuamente a categoria 【TAGLIST】 no painel do browser, o ecrã de acção é aberto.

2Quando toca em [Apagar a lista de tags], todas as faixas são eliminadas da categoria 【TAGLIST】.

Guardar listas de tags como listas de reprodução

Também é possível guardar listas de tags que tenham sido utilizadas para actuações como listas de reprodução.

1Quando prime continuamente a categoria 【TAGLIST】 no painel do browser, o ecrã de acção é aberto.

2Quando toca em [Guardar como lista de reprodução], a caixa de introdução do nome da lista de reprodução é aberta.

3Quando o nome da lista de reprodução é introduzido e toca em [OK], a nova lista de reprodução criada é adicionada à categoria 【PLAYLIST】.

Settings

Preferences

When [Settings] on the top menu or browser panel is tapped, the list of setting items is displayed.

List of setting items

Device Name

Input the name of this mobile device as it is to be displayed on the DJ units (CDJ-2000NXS/CDJ-900NXS/XDJ-1000/XDJ-AERO/XDJ-RX) and in rekordbox (Mac/Windows) on the computer.

Load Lock

This sets whether or not to allow new music files to be loaded when the DJ player onto which you are attempting to load the music files is currently playing.

Track Analysis Mode

This sets the mode for analyzing music files. Select [Normal] for music files with constant tempos, [Dynamic] for music files whose tempo varies.

Key Detection

This sets whether or not to detect the key (pitch) simultaneously when analyzing music files.

Time Mode

This specifies the display mode (Elapsed/Remaining) for the time displayed on the rekordbox (iOS/Android) player panel.

Metronome

This specifies the tone of the metronome sounded when adjusting the beat grid of music files.

  • Beat grid adjustment options

Active Categories

This specifies the items to be displayed in the category menu on the rekordbox (iOS/Android) browser panel and on the DJ players’ main unit display.

  • Setting the active categories and active sort options

Active Sort Options

This specifies the items to be displayed in the sort menu on the DJ players’ main unit display.

  • Setting the active categories and active sort options

User-set category

This specifies the user-set category to be displayed directly to the right of titles when browsing music files on the main unit display of a DJ player (e.g.CDJ-2000NXS).

  • For instructions on browsing libraries on DJ players, refer to the respective DJ players’ operating instructions.

Color

This adds explanations to the color coding (eight colors) used when browsing music files on the main unit display of a DJ player.

  • For instructions on browsing libraries on DJ players, refer to the respective DJ players’ operating instructions.

Arrangement of icons of CDJ/XDJ

The order of the DJ player icons displayed on the link panel (CDJ/XDJ) can be checked and changed according to the actual layout of the DJ players.

  • Performing with your CDJ/XDJ unit(s)

My Settings

A DJ player’s main unit settings can be saved in rekordbox (iOS/Android) and these same settings can be applied to the respective DJ players. This lets users immediately set up DJ players with the settings they are used to so they can devote themselves to DJ performances at their ease.

  • My Settings

Automatic load of Hot Cues to CDJ/XDJ

When selecting music files and loading them onto DJ players, the hot cues (A, B and C) of the DJ player (e.g. CDJ-2000NXS) can be forcibly replaced with the hot cues stored in the loaded music files.

  • Performing with your CDJ/XDJ unit(s)

Library Location (Android devices only)

This specifies the location to which the rekordbox (iOS/Android) library (a database containing the information for the music files, playlists etc.) is stored.

If the rekordbox (iOS/Android) library gets stored in the Android device’s storage area, storage space in the device’s user memory can be saved by changing the location in which the library is stored to the SD memory card.

Cautions

When the library location is changed, previously imported music files can no longer be browsed, so re-import them.

Copy tracks when imported (Android devices only)

This sets whether or not to create copies of music files in the rekordbox (iOS/Android) library when analyzing music files on the Android device and adding them to the rekordbox (iOS/Android) library. Note, however, that if the file path (full path) to the music file has more than 255 characters in it, a copy of the music file is forcibly created in the rekordbox (iOS/Android) library.

  • Using Music Files on a Mobile Device

Purge temporary files

If the connection is cut off or the program is interrupted while importing music files or playlists from rekordbox (Mac/Windows), temporary files will remain in the rekordbox (iOS/Android) library (a database containing the information for the music files, playlists etc.). This function lets you scan for unneeded temporary files and delete them to free up storage space.

Setting the active categories and active sort options

Adding and deleting active categories

1When [Active Categories] is tapped, a list of active categories already set is displayed.

2Tap [Edit], then tap .

A list of the categories that are not displayed opens.

3Tap the category (categories) you want to display. (Multiple categories can be selected.)

4Tap [Done] to close the list of categories that are not displayed.

5Drag on the right edge of the category to change the order in which the active categories are displayed.

An active category can be switched so that it is no longer displayed by tapping on the left of the category then tapping [Inactive].

6Tap [Done] to finish setting the active categories.

Adding and deleting active sort options

The active sort options can also be set using the same type of procedure as described under Adding and deleting active categories.

Importing from a Computer

Playlists and music files that have been analyzed with rekordbox (Mac/Windows) on the computer can be imported to rekordbox (iOS/Android).

Before importing music files, playlists, etc., from rekordbox (Mac/Windows) on the computer, establish the link between the mobile device and computer.

  • Linking with a Computer

1Open the [Device] and [Collection] panes next to each other in rekordbox (Mac/Windows) on the computer.

  • For operations on the computer, see the rekordbox (Mac/Windows) operating instructions.

2Drag the music file from the [Collection] pane to the [Device] pane.

  • For operations on the computer, see the rekordbox (Mac/Windows) operating instructions.

  • The name of the mobile device displayed on the [Device] pane can be checked and changed using rekordbox (iOS/Android).

    • Preferences

The various types of point information, etc., prepared with rekordbox (Mac/Windows) are imported to the rekordbox (iOS/Android) library along with the music files.

Note

To cancel the link with the computer, click at the right of the device icon on the [Device] pane in rekordbox (Mac/Windows).

Importing playlists

Playlists that have been created with rekordbox (Mac/Windows) on the computer can be imported to rekordbox (iOS/Android).

1Open the [Device] and [Playlists] panes next to each other in rekordbox (Mac/Windows) on the computer.

  • For operations on the computer, see the rekordbox (Mac/Windows) operating instructions.

2Drag the playlist from the [Playlists] pane to the [Device] pane.

  • For operations on the computer, see the rekordbox (Mac/Windows) operating instructions.

The various types of point information, etc., prepared with rekordbox (Mac/Windows) are imported to the rekordbox (iOS/Android) library along with the playlists and music files.

Importing “My Settings” values

It is also possible to import the “My Settings” values stored in rekordbox (Mac/Windows) on the computer to rekordbox (iOS/Android).

1In rekordbox (Mac/Windows) on the computer, select [PREFERENCES] from the [File] menu to open the [PREFERENCES] screen.

  • For operations on the computer, see the rekordbox (Mac/Windows) operating instructions.

2At the [PREFERENCES] screen, select [Export] > [My Settings], then open the [My Settings] tab.

  • For operations on the computer, see the rekordbox (Mac/Windows) operating instructions.

3At the [My Settings] tab, click [Apply to all connected devices].

  • For operations on the computer, see the rekordbox (Mac/Windows) operating instructions.

  • When [Apply settings to devices automatically] is checked, the “My Settings” values prepared with rekordbox (Mac/Windows) are automatically imported to rekordbox (iOS/Android) from the next time on.

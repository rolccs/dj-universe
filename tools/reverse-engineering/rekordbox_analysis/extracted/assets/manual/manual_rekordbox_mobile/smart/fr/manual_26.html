Réglages

Mes Réglages

Modèles applicables : CDJ-2000NXS/CDJ-900NXS/XDJ-1000

Les réglages d’appareil principal d’un lecteur DJ peuvent être sauvegardés dans rekordbox (iOS/Android) et ces réglages peuvent être appliqués aux lecteurs DJ respectifs. Ceci permet à l’utilisateur de configurer immédiatement des lecteurs DJ selon ses réglages habituels et de se consacrer entièrement à sa prestation.

1Lorsque [Settings] est tapé dans le menu principal ou sur le panneau d’exploration, la liste des paramètres pouvant être réglés apparaît.

2Lorsque [Mes Réglages] est tapé, les paramètres qui peuvent être réglés dans “Mes Réglages” apparaissent.

3Tapez sur l’option de réglage, puis changez la valeur du réglage.

  • Pour les paramètres pouvant être réglés dans “Mes réglages”, reportez-vous aux modes d’emploi des lecteurs DJ concernés.

4Actualisez les valeurs de “Mes Réglages” sur les différents appareils DJ.

  • Actualisation des valeurs de “Mes Réglages” sur les appareils DJ

Remarques

Il est également possible d’importer dans rekordbox (iOS/Android) les valeurs de “Mes réglages” enregistrées dans rekordbox (Mac/Windows) sur l’ordinateur.

  • Importation depuis un ordinateur

Il est également possible d’exporter les valeurs de “Mes réglages” enregistrées dans rekordbox (iOS/Android) vers rekordbox (Mac/Windows) sur l’ordinateur.

  • Envoi des informations réactualisées à un ordinateur

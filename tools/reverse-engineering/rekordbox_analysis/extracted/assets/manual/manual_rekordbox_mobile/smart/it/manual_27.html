Informazioni aggiuntive

Glossario

BEATGRID

Le battute rilevate quando un file audio viene analizzato vengono visualizzate come una griglia di punti o righe nel display della forma d’onda ingrandita.

MEMORIA DI CUE/LOOP POINT

I punti cue e loop possono venire fissati e memorizzati anticipatamente e poi richiamati coi lettori DJ. Con rekordbox è possibile salvare fino a 10 punti cue e loop per file audio.

STORIA

La storia dei brani riprodotti con lettori DJ può venire restituita a rekordbox. Le storie di riproduzione possono poi essere salvate in rekordbox come playlist.

HOT CUE

Memorizzando cue point e loop point come hot cue, tali punti possono venire raggiunti e riprodotti, mentre la riproduzione normale o di loop può venire iniziata al tocco di un pulsante con un lettore DJ. rekordbox è capace di ricordare fino a tre hot cue per file audio. (* Per XDJ-RX, fino a quattro hot cue possono essere memorizzati.)

LINK EXPORT

Se un lettore DJ ed un computer (o dispositivo mobile) vengono collegati via cavo LAN, LAN wireless o cavo USB, i file e dati musicali di rekordbox possono venire caricati in tempo reale nel lettore DJ.

PLAYLIST

Le playlist possono essere create a seconda delle caratteristiche della performance DJ per organizzare la libreria di musica di rekordbox. Creando playlist di file musicali in anticipo, questi possono venire scelti rapidamente con lettori DJ.

PRO DJ LINK

Collegando fra loro unità DJ Pioneer DJ che supportano PRO DJ LINK via cavo LAN, i file audio di rekordbox possono venire condivise con un mixer DJ e vari lettori DJ. Alla rete possono partecipare fino a 4 lettori DJ. (* Per XDJ-AERO e XDJ-RX, solo un DJ giocatore può partecipare alla rete. XDJ-AERO e XDJ-RX non può partecipare alla stessa rete altri LETTORI DJ.) Se un a rete possiede punti di accesso a reti LAN wireless, ad essa possono partecipare anche 4 dispositivi mobili che possiedono rekordbox (iOS/Android), permettendo di passare senza intoppi da un performer DJ ad un altro.

QUANTIZE

I cue, loop ed hot cue vengono fissati nella battuta più vicina. Usando rekordbox per rilevare le posizioni delle battute di file audio e regolando in anticipo del beatgrid, è possibile impostare cue, loop ed effetti e riprodotti a tempo anche se i pulsanti dell’unità DJ vengono premuti senza precisione.

TAGLIST

I brani scelti a seconda di specifiche situazioni DJ possono venire contrassegnati ed aggiunti a taglist. Le taglist possono venire condivise in linea fra vari lettori DJ collegati via LAN. Controllando i vari lettori DJ, i file audio delle taglist possono venire caricare e quindi riprodotte nei rispettivi lettori DJ, e le taglist usare durante performance possono poi esser salvate come playlist di rekordbox.

  • Per istruzioni su come accedere a taglist da lettori DJ, vedere le istruzioni per l’uso dei lettori DJ.

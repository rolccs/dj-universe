Invio di informazioni aggiornate ad un computer

Nel caso di file audio e playlist importati da rekordbox (Mac/Windows) da un computer a rekordbox (iOS/Android), i dati aggiornati con rekordbox (iOS/Android) possono poi venire inviati a rekordbox (Mac/Windows) del computer.

Prima di trasferire in rekordbox (Mac/Windows) di un computer i dati aggiornati con rekordbox (iOS/Android), stabilire un collegamento fra il dispositivo mobile ed il computer.

  • Collegamento con un computer

1Aprire il riquadro [Dispositivo] in rekordbox (Mac/Windows) nel computer.

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

2Fare un clic destro sull’icona del dispositivo mobile visualizzata in cima al riquadro [Dispositivo].

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

  • Il nome del dispositivo mobile visualizzato nel riquadro [Dispositivo] è controllabile e modificabile con rekordbox (iOS/Android).

    • Preferenze

3Fare clic su [Aggiorna collezione].

  • Per aggiornare i dati un brano alla volta, scegliere il file audio, fare un clic destro e fare clic su [Aggiorna collezione].

Le seguenti voci sono aggiornate nella libreria di rekordbox (Mac/Windows):

  • Memorie di cue/loop point

  • Hot cue

  • Beatgrid

  • Informazioni sui brani (Classifica, Colore, Commenti )

Note

Per esportare una playlist da un dispositivo mobile, trascinarla dal riquadro [Dispositivo] a quello [Playlist].

Per cancellare il collegamento col computer, fare clic su alla destra dell’icona del dispositivo nel riquadro [Dispositivo] in rekordbox (Mac/Windows).

Esportazione dei valori “My Settings”

Si possono anche esportare i valori “My Settings” memorizzati in rekordbox (iOS/Android) in rekordbox (Mac/Windows) di un computer.

1Aprire il riquadro [Dispositivo] in rekordbox (Mac/Windows) nel computer.

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

2Fare un clic destro sull’icona del dispositivo mobile visualizzata nel riquadro [Dispositivo].

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

  • Il nome del dispositivo mobile visualizzato nel riquadro [Dispositivo] è controllabile e modificabile con rekordbox (iOS/Android).

    • Preferenze

3Fare clic su [Importa “MySettings”].

  • I valori “My Settings” memorizzati in rekordbox (iOS/Android) vengono aggiornati anche in rekordbox (Mac/Windows) di un computer.

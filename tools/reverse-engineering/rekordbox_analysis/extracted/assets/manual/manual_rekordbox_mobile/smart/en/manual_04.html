rekordbox Panels

Top Menu

When rekordbox (iOS/Android) is launched, the top menu appears.

Prepare

Used to create the various types of point information, playlists, etc., as advanced preparations for DJ performances.

  • Browser Panel

  • Player Panel

Load

Used to link to DJ players for DJ performances.

  • Linking with CDJ/XDJ Units

  • Linking with an XDJ Unit

Connect

Used to import music files already analyzed with rekordbox (Mac/Windows) on the computer, the various types of point information, playlists, etc., to rekordbox (iOS/Android). Also used to reflect the various types of point information, playlists, etc., that have been updated with rekordbox (iOS/Android) in rekordbox (Mac/Windows).

  • Importing from a Computer

  • Sending Updated Info to a Computer

Import

Used to analyze the music files on the mobile device and add them to the rekordbox (iOS/Android) library.

  • Using Music Files on a Mobile Device

Help

Opens the help menu.

Settings

Used to check and change the various settings.

  • Preferences

Toolbar

See the table below for the icons displayed on the toolbar at the bottom of the screen.

Icon

Description

Menu

Displays the top menu.

Browse

Returns to the uppermost level of the browser panel .

Playing

Used to open the player panel and display the currently playing track.

USB

Used to browse the contents on a USB device connected to the DJ unit (XDJ-AERO) and play them on the DJ unit (XDJ-AERO).

  • For instructions on handling USB devices, see the respective DJ unit’s operating instructions.

Import

Used to analyze the music files on the mobile device and add them to the rekordbox (iOS/Android) library.

  • Using Music Files on a Mobile Device

Settings

Used to check and change the various settings.

  • Preferences

Help

Opens the help file (user’s manual).

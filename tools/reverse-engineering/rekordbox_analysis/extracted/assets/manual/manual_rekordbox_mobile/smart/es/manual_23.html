Selección de música en una lista de fichas

La lista de fichas es una lista que se puede compartir en tiempo real por múltiples reproductores DJ conectados a una LAN.

1Cuando toca y mantiene tocada una pista del panel explorador, la hoja de acción se abre.

2Cuando se toca [Añadir a lista de fichas], la pista seleccionada se añade a la categoría 【TAGLIST】.

  • Para instrucciones de acceso a listas de fichas desde reproductores DJ, consulte los manuales de instrucciones de los reproductores DJ respectivos.

Eliminación de pistas de listas de fichas

Para eliminar una pista cada vez

1En el panel explorador, toque la categoría 【TAGLIST】.

Se visualiza una lista de pistas que ha sido añadida a la lista de fichas.

2Cuando toca y mantiene tocada la pista que quiere eliminar, la hoja de acción se abre.

3Cuando se toca [Quitar de lista de fichas], la pista seleccionada se elimina de la categoría 【TAGLIST】.

Para eliminar todas las pistas

1Cuando toca y mantiene tocada la categoría 【TAGLIST】 del panel explorador, la hoja de acción se abre.

2Cuando se toca [Borrar la lista de fichas], todas las pistas se eliminan de la categoría 【TAGLIST】.

Para guardar listas de fichas y listas de reproducción

También es posible guardar listas de fichas que han sido usadas en actuaciones como listas de reproducción.

1Cuando toca y mantiene tocada la categoría 【TAGLIST】 del panel explorador, la hoja de acción se abre.

2Cuando se toca [Guardar como lista de reproducción], la casilla de introducción de nombres de listas de reproducción se abre.

3Cuando se introduce el nombre de la lista de reproducción y luego se toca [OK], la lista de reproducción recién creada se añade a la categoría 【PLAYLIST】.

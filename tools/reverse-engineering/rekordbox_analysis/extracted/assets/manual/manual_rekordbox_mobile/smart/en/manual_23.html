DJ Performances

Selecting Music for a Taglist

The tag list is a list that can be shared in real time among multiple DJ players connected on a LAN.

1When a track on the browser panel is touched and held, the action sheet opens.

2When [Add to Tag List] is tapped, the selected track is added to the 【TAGLIST】 category.

  • For instructions on accessing taglists from DJ players, see the operating instructions for the respective DJ players.

Deleting tracks from taglists

To delete one track at a time

1On the browser panel, tap the 【TAGLIST】 category.

A list of the tracks that have been added to the taglist is displayed.

2When the track you want to delete is touched and held, the action sheet opens.

3When [Remove from Tag List] is tapped, the selected track is deleted from the 【TAGLIST】 category.

To delete all the tracks

1When the 【TAGLIST】 category on the browser panel is touched and held, the action sheet opens.

2When [Clear Tag List] is tapped, all the tracks are deleted from the 【TAGLIST】 category.

Saving taglists as playlists

It is also possible to save taglists that have been used for performances as playlists.

1When the 【TAGLIST】 category on the browser panel is touched and held, the action sheet opens.

2When [Save as Playlist] is tapped, the playlist name input box opens.

3When the playlist name is input then [OK] is tapped, the newly created playlist is added to the 【PLAYLIST】 category.

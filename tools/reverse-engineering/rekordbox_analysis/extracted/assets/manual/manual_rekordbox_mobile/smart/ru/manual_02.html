Что такое rekordbox?

rekordbox™ является программным обеспечением, позволяющим пользователям, которые приобрели DJ изделие Пайонир Диджей, управлять музыкальными файлами, которые они используют во время DJ исполнений.

  • rekordbox может использоваться для обнаружения, измерения и регулировки позиций ударов, темпов (ВРМ), др. музыкальных файлов.

  • rekordbox может использоваться для классификации и поиска музыкальных файлов и создания списков воспроизведения, др., для различных ситуаций DJ исполнения.

  • rekordbox может использоваться для установки и сохранения подробной информации точек (для меток, петлей, меток быстрого доступа, др.) до исполнения.

Выполнение приготовлений с помощью мобильного устройства

Музыкальные файлы, проанализированные с помощью rekordbox (Mac/Windows) на компьютере, списки воспроизведения, др. можно загрузить на rekordbox (iOS/Android) на мобильном устройстве. Также, информацию,обновленную с помощью rekordbox (iOS/Android), можно возвратить на rekordbox (Mac/Windows) на компьютере.

Подключение к DJ изделиям для DJ исполнений

Для выполнения DJ исполнений можно подключить DJ изделия Пайонир Диджей, поддерживающие rekordbox (iOS/Android) (CDJ-2000NXS/CDJ-900NXS/XDJ-1000/XDJ-AERO/XDJ-RX). С помощью контактной панели можно легко выполнять поиск музыкальных файлов на мобильных устройствах и загружать их на DJ проигрыватель или деку.

Обратная связь с историей воспроизведения

Историю загружавшихся на DJ проигрыватели и воспроизводившихся музыкальных файлов можно сохранить в виде списков воспроизведения, которые могут использоваться для будущих DJ исполнений.

Performances DJ

Performance con un’unità XDJ propria

Modelli di applicabilità: XDJ-AERO/XDJ-RX

I file audio possono essere scelti nel pannello del browser e caricati in unità DJ.

1Stabilire il collegamento fra rekordbox (iOS/Android) e l’unità DJ.

  • Collegamento con una unità XDJ

Stabilito il collegamento, il pannello del browser si apre.

2Quando un brano viene toccato nel pannello del browser, il pannello di collegamento (di XDJ) si apre.

Le playlist e le taglist possono essere usate per organizzare i file audio per essere convenienti nello scegliere i brani.

  • Organizzazione di musica con Playlist

  • Scelta di musica per una taglist

3Toccando l’icona del deck dell’unità DJ, il file musicale viene caricato nel deck.

  • Quando la funzione AUTOMIX dell’unità DJ (XDJ-AERO) è accesa ed il file audio viene scelto da una playlist e caricato, il file audio successivo viene a sua volta caricato automaticamente nell’altro deck.

Nota

Il contenuto di un dispositivo di memoria di massa USB collegato all’unità DJ (XDJ-AERO) può essere esplorato e scelto toccando [USB] nella toolbar del pannello del browser.

  • Per istruzioni sull’uso di dispositivi USB, vedere le istruzioni per l’uso della rispettiva unità DJ.

Pour les utilisateurs novices

Avant toute chose, préparez les morceaux qui doivent être utilisés pour votre prestation DJ. Analysez les fichiers de musique présents sur le dispositif portable et ajoutez-les à la bibliothèque rekordbox (iOS/Android).

  • Utilisation de fichiers de musique sur un dispositif portable

Si rekordbox (Mac/Windows) est déjà installé sur votre ordinateur, les fichiers de musique déjà analysés avec rekordbox (Mac/Windows), les listes de lecture, etc., pourront être importés sur rekordbox (iOS/Android).

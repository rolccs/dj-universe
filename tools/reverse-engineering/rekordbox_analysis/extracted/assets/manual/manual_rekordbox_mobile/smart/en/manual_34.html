XDJ-AERO Connection Guide

In order to establish communications between a mobile device and the XDJ-AERO, the XDJ-AERO’s SSID and password must first be checked.

Before using the XDJ-AERO, carefully read the “Read Before Use” booklet and the “Operating Instructions” CD-ROM included with the XDJ-AERO.

1When the XDJ-AERO’s [INFO/UTILITY] button is pressed for over 1 second, the [UTILITY] menu appears on the XDJ-AERO’s main unit display.

2Turn the XDJ-AERO’s rotary encoder to select [WLAN INFORMATION], then press the rotary encoder.

3When the XDJ-AERO’s rotary encoder is turned to select [SSID] then the rotary encoder is pressed, the XDJ-AERO’s SSID is displayed.

4Press the XDJ-AERO’s [BACK/TOP] button to return to the previous display.

5When the XDJ-AERO’s rotary encoder is turned to select [PASSWORD] then the rotary encoder is pressed, the XDJ-AERO’s password is displayed.

6Open the mobile device’s Wi-Fi settings screen, tap the XDJ-AERO’s SSID, then input the password.

7Launch rekordbox on the mobile device, then tap [Load] on the top menu.

When the link between rekordbox and the XDJ-AERO is established, the indicator at the top left of the XDJ-AERO lights.

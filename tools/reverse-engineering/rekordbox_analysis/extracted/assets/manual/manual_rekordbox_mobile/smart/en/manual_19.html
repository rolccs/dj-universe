Preparation Work

Loops

When a loop is set after turning the quantize function on, the point is set to the nearest beat position on the beat grid. (Beat Snap)

1Play the track on the player panel.

2When [IN] is tapped, the currently playing position is set as the loop in point.

  • This operation is not necessary when setting an already set cue point as the loop in point.

3When [OUT] is tapped, the currently playing position is set as the loop out point.

  • The track returns to the loop in point and loop playback starts.

Notes

When new loop points are set, the previously set loop points are deleted. Save loop points you want to keep.

  • Saving loops (Cue/Loop Point Memory)

When a different music file is selected, loop points that have not been saved are canceled.

Canceling/resuming loop playback (Loop Exit/Reloop)

Loop Exit

When [RELOOP/EXIT] is tapped during loop playback, playback continues when the loop out point is reached, without returning to the loop in point.

Reloop

When [RELOOP/EXIT] is tapped again after canceling loop playback, the track returns to the loop in point and loop playback resumes.

Specifying the loop length by the number of beats (Auto Beat Loop)

A loop with the specified number of beats is set and loop playback starts.

1Play the track on the player panel.

2When [4], [8], [16] or [32] is tapped, the currently playing position is set as the loop in point.

The loop out point is set at the position corresponding to the set number of beats, and when it is reached, the track returns to the loop in point and loop playback starts.

3When [4], [8], [16] or [32] is tapped during loop playback, only the loop out point is reset.

Saving loops (Cue/Loop Point Memory)

A total of up to 10 cue and loop points can be saved per music file.

Saving set loops

When [MEMORY] is tapped during loop playback, the loop points are saved and indicated [] at the top of the waveform display.

Calling out saved loops

When [CALL ] or [CALL ] is tapped, the track is cued at the saved loop in point and the pause mode is set.

  • Tap [CALL ] to call out a point before the currently playing position, [CALL ] to call out a point after the currently playing position.

  • When [] is tapped, playback starts from the loop point that was called out.

  • When [DEL] is tapped, the loop point that was called out is deleted.

Saving saved loops as the active loop

When a saved looped is called out, the [MEMORY] display switches to [ACTIVE].

To save a saved loop as the active loop, tap [ACTIVE].

  • Only one loop can be saved as the active loop. When the loop that was saved as the active loop is called out, [MEMORY/ACTIVE] is displayed in red. To cancel the active loop, click [ACTIVE] again.

  • Loops that have been saved as the active loop cannot be played as the active loop from the rekordbox (iOS/Android) player panel. For instructions on using the active loop, see the operating instructions of the respective DJ player (e.g. CDJ-2000NXS). When the playing position reaches the point set as the active loop, loop playback begins automatically.

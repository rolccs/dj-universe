Envoi des informations réactualisées à un ordinateur

En ce qui concernent les fichiers de musique et les listes de lecture importés dans rekordbox (iOS/Android) depuis rekordbox (Mac/Windows) sur l’ordinateur, leurs informations mises à jour à l’aide de rekordbox (iOS/Android) peuvent être mises à jour dans rekordbox (Mac/Windows) sur l’ordinateur.

Établissez une liaison entre le dispositif portable et l’ordinateur avant de renvoyer les informations mises à jour à l’aide de rekordbox (iOS/Android) à rekordbox (Mac/Windows) sur l’ordinateur.

  • Liaison à un ordinateur

1Ouvrez le volet [Dispositif] dans rekordbox (Mac/Windows) sur l’ordinateur.

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

2Cliquez à droite sur l’icône du dispositif portable au haut du volet [Dispositif].

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

  • Le nom du dispositif portable apparaissant dans le volet [Dispositif] peut être vérifié et changé à l’aide de rekordbox (iOS/Android).

    • Préférences

3Cliquez sur [Mettre la collection à jour].

  • Pour réactualiser chaque morceau individuellement, sélectionnez le fichier de musique, cliquez à droite, puis cliquez sur [Mettre la collection à jour].

Les éléments suivants sont réactualisés dans la bibliothèque rekordbox (Mac/Windows) :

  • Mémoires de points de repère/boucles

  • Repères instantanés

  • Grille rythmique

  • Informations des morceaux (Classement, Couleur, Commentaires)

Remarques

Pour exporter une liste de lecture du dispositif portable, tirez-la du volet [Dispositif] pour la déposer dans le volet [Liste de lecture].

Pour annuler la liaison avec l’ordinateur, cliquez sur à la droite de l’icône de dispositif sur le volet [Dispositif] dans rekordbox (Mac/Windows).

Exportation des valeurs de “Mes réglages”

Il est également possible d’exporter les valeurs de “Mes réglages” enregistrées dans rekordbox (iOS/Android) vers rekordbox (Mac/Windows) sur l’ordinateur.

1Ouvrez le volet [Dispositif] dans rekordbox (Mac/Windows) sur l’ordinateur.

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

2Cliquez à droite sur l’icône du dispositif portable apparaissant sur le volet [Dispositif].

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

  • Le nom du dispositif portable apparaissant dans le volet [Dispositif] peut être vérifié et changé à l’aide de rekordbox (iOS/Android).

    • Préférences

3Cliquez sur [Importer « Mes Réglages »].

  • Les valeurs de “Mes réglages” enregistrées dans rekordbox (iOS/Android) sont renvoyées à rekordbox (Mac/Windows) sur l’ordinateur.

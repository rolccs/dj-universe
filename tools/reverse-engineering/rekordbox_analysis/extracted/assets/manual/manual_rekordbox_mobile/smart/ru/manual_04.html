Панели rekordbox

Главное меню

При запуске rekordbox (iOS/Android) отображается главное меню.

Prepare

Используется для создания различных типов информации по точкам, списков воспроизведения, др. в качестве дополнительной подготовки к DJ исполнениям.

  • Панель просмотра

  • Панель проигрывателя

Load

Используется для подключения к DJ проигрывателям для выполнения DJ исполнений.

  • Подключение к CDJ/XDJ изделиям

  • Подключение к XDJ изделию

Connect

Используется для импорта музыкальных файлов, уже проанализированных на rekordbox (Mac/Windows) на компьютере, различных типов информации по точкам, списков воспроизведения, др. на rekordbox (iOS/Android). Также используется для отражения различных типов информации по точкам, списков воспроизведения, др., которые были обновлены с помощью rekordbox (iOS/Android) в rekordbox (Mac/Windows).

  • Импорт с компьютера

  • Передача обновленной информации на компьютер

Import

Используется для анализа музыкальных файлов на мобильном устройстве и их добавления в библиотеку rekordbox (iOS/Android).

  • Использование музыкальных файлов на мобильном устройстве

Help

Открывает меню помощи.

Settings

Используется для проверки и изменения различных настроек.

  • Предпочтения

Панель инструментов

В таблице ниже указаны иконки, отображающиеся в панели инструментов в нижней части экрана.

Иконка

Описание

Menu

Отображает главное меню.

Browse

Возвращает на самый вехний уровень панели просмотра .

Playing

Используется для открытия панели проигрывателя и отображаения текущей воспроизводящейся дорожки.

USB

Используется для просмотра материала на устройстве USB, подключенном к DJ изделию (XDJ-AERO), и для воспроизведения такого материала на DJ изделии (XDJ-AERO).

  • Подробнее об использовании устройств USB смотрите инструкции по эксплуатации к соответствующему DJ изделию.

Import

Используется для анализа музыкальных файлов на мобильном устройстве и их добавления в библиотеку rekordbox (iOS/Android).

  • Использование музыкальных файлов на мобильном устройстве

Settings

Используется для проверки и изменения различных настроек.

  • Предпочтения

Help

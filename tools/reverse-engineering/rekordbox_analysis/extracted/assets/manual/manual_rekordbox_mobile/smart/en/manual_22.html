DJ Performances

Performing with your XDJ Unit

Applicable models: XDJ-AERO/XDJ-RX

Music files can be selected on the browser panel and loaded onto DJ unit.

1Establish the link between rekordbox (iOS/Android) and the DJ unit.

  • Linking with an XDJ Unit

When the link is established, the browser panel opens.

2When a track is tapped on the browser panel, the link panel (XDJ) opens.

Playlists and taglists can be used to organize music files for convenience when selecting tracks.

  • Organizing Music in Playlists

  • Selecting Music for a Taglist

3When a DJ unit’s deck icon is tapped, the music file is loaded onto that deck.

  • When the DJ unit’s (XDJ-AERO) AUTOMIX function is on and a music file is selected from a playlist and loaded, the next music file is also automatically loaded into the other deck.

Note

The contents of the USB device connected to the DJ unit (XDJ-AERO) can be browsed and selected by tapping [USB] on the toolbar on the browser panel.

  • For instructions on handling USB devices, see the respective DJ unit’s operating instructions.

Music Management

Browsing Music Files

Music files can be browsed and selected using the track’s information (such attributes as the category, etc.).

If no tracks or categories are displayed on the browser panel, first analyze the music files and add them to the rekordbox (iOS/Android) library.

  • For First-time Users

1When [Prepare] is tapped at the top menu, the browser panel opens.

2Tap the category you want to browse.

The next level in the selected category is displayed.

  • It is also possible to add categories to be displayed on the browser panel.

    • Preferences

  • For such categories as 【TRACK】, 【ALBUM】 and 【ARTIST】, when the index bar on the right edge of the browser panel is tapped, the list scrolls to the line that starts with the selected letter.

3Select the track and tap it.

The player panel appears and the track is played.

When linked to DJ players, the link panel (CDJs/XDJs) or link panel (XDJ) opens.

  • To move back to the level above, tap .

Searching for tracks

Tracks in the rekordbox (iOS/Android) library can be searched for by inputting character strings.

1When you swipe downwards on the browser panel and return to the top of the list, a search box appears at the very top of the list.

  • If the search box does not appear when this operation is performed on your Android device, press the Android device’s menu button, then tap [Search].

2Tap the search box and input the character string you want to search for.

Artists, albums and tracks including the character string you have input are displayed.

Deleting tracks

The tracks are deleted from the rekordbox (iOS/Android) library.

1When the track you want to delete is touched and held, the action sheet opens.

2Tap [Remove from rekordbox] to delete the track from the library.

Deleting albums and artists

When an album or artist is touched and held then [Remove from rekordbox] is tapped, all the tracks included in that album or artist category are removed from the rekordbox (iOS/Android) library.

Editing track information

1When the track whose information you want to edit is touched and held, the action sheet opens.

2When [Info] is tapped, the track information is displayed.

Items that can be edited

Rating, color and comments

  • The track information edited in rekordbox (iOS/Android) can be reflected in the library of rekordbox (Mac/Windows) on the computer.

    • Sending Updated Info to a Computer

Reanalyzing music files

1When the track you want to reanalyze is touched and held, the action sheet opens.

2Tap [Analyze Track] to start analyzing the music files.

Once analysis of the music file has finished, the BPM and beat grids are overwritten with the newly analyzed results.

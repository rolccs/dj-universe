Передача обновленной информации на компьютер

Для музыкальных файлов и списков воспроизведения, импортированных с rekordbox (Mac/Windows) на компьютере в rekordbox (iOS/Android), информацию, обновленную с помощью rekordbox (iOS/Android), можно обновить и в rekordbox (Mac/Windows) на компьютере.

Перед отражением информации, обновленной с помощью rekordbox (iOS/Android), в rekordbox (Mac/Windows) на компьютере требуется установить связь между мобильным устройством и компьютером.

  • Подключение к компьютеру

1Откройте окно [Устройство] в rekordbox (Mac/Windows) на компьютере.

  • Подробнее об операциях на компьютере смотрите инструкции по эксплуатации к rekordbox (Mac/Windows).

2Щелкните правой клавишей по иконке мобильного устройства, отображенной в верхней части окна [Устройство].

  • Подробнее об операциях на компьютере смотрите инструкции по эксплуатации к rekordbox (Mac/Windows).

  • Отображаемое в окне [Устройство] имя мобильного устройства можно проверить и отредактировать с помощью rekordbox (iOS/Android).

    • Предпочтения

3Щелкните по [Обновить сборник].

  • Для обновления одной дорожки за раз выберите музыкальный файл, щелкните правую клавишу, затем щелкните по [Обновить сборник].

В библиотеке rekordbox (Mac/Windows) обновляются следующие параметры:

  • Память точек меток/петель

  • Метки быстрого доступа

  • Сетка ударов

  • Информация дорожки (Рейтинг, Цвет, Комментарии)

Примечания

Для экспорта списка воспроизведения с мобильного устройства перетащите список воспроизведения с окна [Устройство] в окно [Список воспроизведения].

Для отмены подключения к компьютеру щелкните по справа от иконки устройства на окне [Устройство] в rekordbox (Mac/Windows).

Экспорт значений “Мои настройки”

Также можно экспортировать значения “Мои настройки”, сохраненные в rekordbox (iOS/Android), в rekordbox (Mac/Windows) на компьютере.

1Откройте окно [Устройство] в rekordbox (Mac/Windows) на компьютере.

  • Подробнее об операциях на компьютере смотрите инструкции по эксплуатации к rekordbox (Mac/Windows).

2Щелкните правой клавишей по иконке мобильного устройства, отображенной на окне [Устройство].

  • Подробнее об операциях на компьютере смотрите инструкции по эксплуатации к rekordbox (Mac/Windows).

  • Отображаемое в окне [Устройство] имя мобильного устройства можно проверить и отредактировать с помощью rekordbox (iOS/Android).

    • Предпочтения

3Щелкните по [Импортировать «Мои настройки»].

  • Значения “Мои настройки”, сохраненные в rekordbox (iOS/Android), отражаются в rekordbox (Mac/Windows) на компьютере.

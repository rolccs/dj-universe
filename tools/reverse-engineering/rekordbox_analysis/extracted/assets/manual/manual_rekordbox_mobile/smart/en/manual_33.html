CDJ-2000NXS Connection Guide

In order to establish communications between a mobile device and the CDJ-2000NXS, the SSID and password of the wireless LAN (Wi-Fi) access point to which the CDJ-2000NXS is connected must first be checked.

Before using the CDJ-2000NXS, carefully read the “Read Before Use” booklet and the “Operating Instructions” CD-ROM included with the CDJ-2000NXS.

1Connect the wireless LAN (Wi-Fi) access point and the CDJ-2000NXS by LAN cable.

2Check the SSID and password indicated on the wireless LAN (Wi-Fi) access point, in its instructions, etc.

3Open the mobile device’s Wi-Fi settings screen, tap the access point’s SSID, then input the password.

4Launch rekordbox on the mobile device, then tap [Load] on the top menu.

  • When connectable DJ players are found, the message “Please select your smartphone on the screen of the CDJ/XDJ” appears.

5When the CDJ-2000NXS’s [rekordbox] button is pressed, the list of connectable mobile devices appears on the CDJ-2000NXS’s main unit display.

6Turn the CDJ-2000NXS’s rotary encoder to select the mobile device, then press the rotary encoder.

Paneles del rekordbox

Menú inicial

Cuando se inicia el rekordbox (iOS/Android) aparece el menú inicial.

Prepare

Se usa para crear varios tipos de información de puntos, listas de reproducción, etc., como preparativos avanzados para actuaciones de DJ.

  • Panel explorador

  • Panel del reproductor

Load

Se usa para enlazar reproductores DJ para actuaciones de DJ.

  • Enlace con unidades CDJ/XDJ

  • Enlace con una unidad XDJ

Connect

Se usa para importar archivos de música ya analizados con el rekordbox (Mac/Windows) del ordenador, varios tipos de información de puntos, listas de reproducción, etc., al rekordbox (iOS/Android). También se usa para reflejar varios tipos de información de puntos, listas de reproducción, etc., que han sido actualizados con el rekordbox (iOS/Android) en el rekordbox (Mac/Windows).

  • Importación de un ordenador

  • Envío de información actualizada a un ordenador

Import

Se usa para analizar los archivos de música del aparato portátil y añadirlos a la librería del rekordbox (iOS/Android).

  • Uso de los archivos de música en un aparato portátil

Help

Abra el menú de ayuda.

Se usa para verificar y cambiar los diversos ajustes.

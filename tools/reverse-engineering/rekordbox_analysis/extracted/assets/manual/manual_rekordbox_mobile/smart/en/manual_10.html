Linking with CDJ/XDJ Units

Applicable models: CDJ-2000NXS/CDJ-900NXS/XDJ-1000/XDJ-RX

Mobile devices can be connected to Pioneer DJ players supporting rekordbox (iOS/Android) via a wireless LAN device to exchange music data.

1Connect the DJ player to a network with a wireless LAN access point (PRO DJ LINK).

  • For details on the wireless LAN connection types and connection procedure, see the respective DJ player’s operating instructions and your wireless LAN device’s instructions.

  • CDJ-2000NXS Connection Guide

2Connect the mobile device on which rekordbox (iOS/Android) is installed to the wireless LAN network.

  • For instructions on making the wireless LAN connections, see the instructions of your mobile device and wireless LAN device.

3On rekordbox (iOS/Android)’s top menu, tap [Load].

When a DJ player with which communications can be established is found, rekordbox (iOS/Android) stands by for the connection operation (link authorization) to be performed on the DJ player.

4On the DJ player, specify the mobile device whose link is to be authorized.

  • For the operation of authorizing linking, see the respective DJ players’ operating instructions.

  • CDJ-2000NXS Connection Guide

When the link is established, the browser panel opens.

Notes

Up to four mobile devices can be linked simultaneously to the DJ player. (* For XDJ-RX, up to two mobile devices can be linked.)

The name of the mobile device displayed on the DJ player can be checked and changed using rekordbox (iOS/Android).

  • Preferences

Connecting to a DJ player with a USB cable

iOS devices (such as iPhones) and Android devices supporting the MTP mode (Android 3.0 and later) can be connected to the DJ player using a USB cable.

After connecting the mobile device and DJ player using a USB cable, tap [Load] on the rekordbox (iOS/Android) top menu . Once the link is established, the browser panel opens.

  • For instructions on setting the MTP mode, restrictions in operations from the Android device, communications restrictions, etc., see your Android device’s instructions.

  • Android devices set to the MSC mode operate as USB storage devices (secondary storage devices such as USB flash memory drives) when connected to a DJ player by USB cable. In this case, data cannot be transferred using rekordbox (iOS/Android). For details on connections between DJ players and USB storage devices, see the respective DJ player’s operating instructions.

Canceling the link with the DJ player

On the browser panel, when you tap or [Menu] on the toolbar then return to the top menu, the link to the DJ player is canceled.

Starting Up

What’s rekordbox?

rekordbox™ is a software program allowing users who have purchased a Pioneer DJ unit to manage the music files they use for DJ performances.

  • rekordbox can be used to detect, measure and adjust the beat positions, tempos (BPMs), etc., of music files.

  • rekordbox can be used to classify and search for music files and create playlists, etc., for different DJing situations.

  • rekordbox may be used to set and store detailed point information (for cueing, looping, hot cueing, etc.) prior to a performance.

Making preparations using a mobile device

Music files that have been analyzed with rekordbox (Mac/Windows) on a computer, playlists, etc., can be loaded into rekordbox (iOS/Android) on a mobile device. Also, information that has been updated using rekordbox (iOS/Android) can be returned to rekordbox (Mac/Windows) on a computer.

Connecting to DJ units for DJ performances

Pioneer DJ units supporting rekordbox (iOS/Android) (CDJ-2000NXS/CDJ-900NXS/XDJ-1000/XDJ-AERO/XDJ-RX) can be connected to hold DJ performances. The music files on mobile devices can be browsed comfortably using the touch panel and loaded into the DJ player or deck.

Feedback of the playing history

The history of music files that have been loaded onto DJ players and played can be stored as playlists that can be used for future DJ performances.

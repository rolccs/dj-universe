Settings

My Settings

Applicable models: CDJ-2000NXS/CDJ-900NXS/XDJ-1000

A DJ player’s main unit settings can be saved in rekordbox (iOS/Android) and these same settings can be applied to the respective DJ players. This lets users immediately set up DJ players with the settings they are used to so they can devote themselves to DJ performances at their ease.

1When [Settings] is tapped at the top menu or on the browser panel, the list of setting items is displayed.

2When [My Settings] is tapped, the items that can be set at “My Settings” are displayed.

3Tap the setting option, then change the setting’s value.

  • For the items that can be set at “My Settings”, see the respective DJ player’s operating instructions.

4Reflect the “My Settings” values on the respective DJ players.

  • Reflecting the “My Settings” values on the DJ units

Notes

It is also possible to import the “My Settings” values stored in rekordbox (Mac/Windows) on the computer to rekordbox (iOS/Android).

  • Importing from a Computer

It is also possible to export the “My Settings” values stored in rekordbox (iOS/Android) to rekordbox (Mac/Windows) on the computer.

XDJ-AERO Guide de raccordement

Pour établir des communications entre un dispositif portable et le XDJ-AERO, le SSID et le mot de passe du XDJ-AERO doivent d’abord être vérifiés.

Avant d’utiliser le XDJ-AERO, lisez attentivement la brochure “A lire avant l’utilisation” et le “Mode d’emploi” sur le CD-ROM fourni avec le XDJ-AERO.

1Lorsque la touche [INFO/UTILITY] du XDJ-AERO est pressée plus d’une seconde, le menu [UTILITY] apparaît sur l’afficheur principal du XDJ-AERO.

2Tournez le codeur rotatif du XDJ-AERO pour sélectionner [WLAN INFORMATION], puis appuyez sur le codeur rotatif.

3Lorsque le sélecteur rotatif du XDJ-AERO est tourné pour sélectionner [SSID], puis pressé, le SSID du XDJ-AERO apparaît.

4Appuyez sur la touche [BACK/TOP] du XDJ-AERO pour revenir à l’affichage précédent.

5Lorsque le codeur rotatif du XDJ-AERO est tourné pour sélectionner [PASSWORD], puis pressé, le mot de passe du XDJ-AERO apparaît.

6Ouvrez l’écran des réglages Wi-Fi du dispositif portable, tapez sur le SSID du XDJ-AERO, puis saisissez le mot de passe.

7Lancez rekordbox sur le dispositif portable, puis tapez sur [Load] sur le menu principal.

Lorsque la liaison entre rekordbox et le XDJ-AERO est établie, l’indicateur dans le coin supérieur gauche de XDJ-AERO s’allume.

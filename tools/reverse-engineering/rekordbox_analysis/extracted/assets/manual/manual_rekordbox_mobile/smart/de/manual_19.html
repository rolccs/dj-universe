Vorbereitungsarbeiten

Loops

Wird ein Loop nach dem Einschalten der Quantisierungsfunktion eingestellt, wird der Punkt auf die nächstliegende Beat-Position auf dem Beatgrid eingestellt. (Beat Snap)

1Spielen Sie den Track auf dem Player-Felds ab.

2Wenn [IN] angetippt wird, wird die aktuell spielende Position als Loop-In-Punkt gesetzt.

  • Dieser Vorgang ist nicht erforderlich, wenn Sie einen bereits gesetzten Cue-Punkt als Loop-In-Punkt setzen.

3Wenn [OUT] angetippt wird, wird die aktuell spielende Position als Loop-Out-Punkt gesetzt.

  • Der Track kehrt sofort zum Loop-in-Punkt zurück, und die Loop-Wiedergabe startet.

Hinweise

Wenn neue Loop-Punkte gesetzt werden, werden die vorher gesetzten Loop-Punkte gelöscht. Speichern Sie Loop-Punkte, die Sie bewahren wollen.

  • Speichern von Loops (Cue/Loop-Punkt-Speicher)

Wenn ein andere Musikdatei gewählt wird, werden nicht gespeicherte Loop-Punkte aufgehoben.

Abbrechen/Fortsetzen der Loop-Wiedergabe (Loop Exit/Reloop)

Loop Exit

Wenn [RELOOP/EXIT] während der Loop-Wiedergabe angetippt wird, wird die Wiedergabe fortgesetzt, wenn der Loop-Out-Punkt erreicht ist, ohne dass zum Loop-In-Punkt zurückgegangen wird.

Reloop

Wenn [RELOOP/EXIT] erneut angetippt wird, nachdem die Loop-Wiedergabe abgebrochen wird, kehrt der Track zum Loop-In-Punkt zurück, und die Loop-Wiedergabe wird fortgesetzt.

Festlegen der Loop-Länge in Anzahl der Beats (Auto Beat Loop)

Ein Loop mit der festgelegten Anzahl der Beats wird eingestellt, und die Loop-Wiedergabe beginnt.

1Spielen Sie den Track auf dem Player-Felds ab.

2Wenn [4], [8], [16] oder [32] angetippt wird, wird die aktuell spielende Position als Loop-In-Punkt gesetzt.

Der Loop-Out-Punkt wird bei der Position gesetzt, die der eingestellten Anzahl von Beats entspricht, und wenn er erreicht ist, kehrt der Track zum Loop-in-Punkt zurück, und die Loop-Wiedergabe beginnt.

3Wenn [4], [8], [16] oder [32] während der Loop-Wiedergabe angetippt wird, wird nur der Loop-Out-Punkt zurückgesetzt.

Speichern von Loops (Cue/Loop-Punkt-Speicher)

Bis zu 10 Cue- und Loop-Punkte können pro Musikdatei gespeichert werden.

Speichern von gesetzten Loops

Wenn [MEMORY] während der Loop-Wiedergabe angetippt wird, werden die Loop-Punkte gespeichert und [] wird oben in der Wellenform-Anzeige angezeigt.

Aufrufen gespeicherter Loops

Wenn [CALL ] oder [CALL ] angetippt wird, wird der Track beim gespeicherten Loop-In-Punkt gecued, und der Pause-Modus wird eingestellt.

  • Tippen Sie [CALL ] an, um einen Cue-Punkt vor der aktuell spielenden Position aufzurufen und [CALL ], um einen Punkt nach der aktuell spielenden Position aufzurufen.

  • Wenn [] angetippt wird, beginnt die Wiedergabe von dem Loop-Punkt, der aufgerufen wurde.

  • Wenn [DEL] angetippt wird, wird der aufgerufene Loop-Punkt gelöscht.

Speichern gespeicherter Loops als aktiver Loop

Wenn ein gespeicherter Loop aufgerufen wird, schaltet die [MEMORY]-Anzeige auf [ACTIVE] um.

Zum Speichern eines gespeicherten Loops als aktiver Loop tippen Sie auf [ACTIVE].

  • Nur ein Loop kann als aktiver Loop gespeichert werden. Wenn der als aktiver Loop gespeicherte Loop aufgerufen wird, wird [MEMORY/ACTIVE] in rot angezeigt. Zum Abbrechen des aktiven Loops klicken Sie erneut auf [ACTIVE].

  • Loops, die als aktiver Loop gespeichert wurden, können nicht als aktiver Loop vom Player-Feld von rekordbox (iOS/Android) gespielt werden. Anweisungen zur Verwendung des aktiven Loops siehe Bedienungsanleitung des betreffenden DJ-Players (z.B. CDJ-2000NXS). Wenn die Wiedergabeposition den als aktiven Loop eingestellten Punkt erreicht, beginnt die Loop-Wiedergabe automatisch.

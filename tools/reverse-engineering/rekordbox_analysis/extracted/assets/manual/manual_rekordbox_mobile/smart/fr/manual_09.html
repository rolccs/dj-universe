Connexions

Liaison à un ordinateur

Le dispositif portable peut se connecter par un réseau sans fil à un ordinateur sur lequel rekordbox (Mac/Windows) a été installé pour échanger des données de musique avec rekordbox (Mac/Windows).

Selon les réglages de l’antivirus, le système d’exploitation, etc. utilisés sur l’ordinateur, la communication entre l’ordinateur et le dispositif portable peut être interrompue. Pour le détail sur l’environnement requis sur l’ordinateur pour la communication (programmes, système d’exploitation, réseau, etc.), reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

1Connectez l’ordinateur sur lequel rekordbox (Mac/Windows) est installé au réseau sans fil.

  • Pour les instructions sur la réalisation de connexions LAN sans fil, reportez-vous aux instructions de votre ordinateur ou du dispositif sans fil.

2Connectez le dispositif portable sur lequel rekordbox (iOS/Android) est installé au réseau sans fil.

  • Pour les instructions sur la réalisation de connexions LAN sans fil, reportez-vous aux instructions de votre dispositif portable et du dispositif sans fil.

3Lancez rekordbox (Mac/Windows) sur l’ordinateur.

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

4Sur le menu principal de rekordbox (iOS/Android), tapez sur [Connect].

rekordbox (iOS/Android) attend qu’une liaison (recherche de dispositif portable) soit effectuée dans rekordbox (Mac/Windows) sur l’ordinateur.

5Ouvrez le volet [Dispositif] dans rekordbox (Mac/Windows) sur l’ordinateur et cliquez sur dans le coin inférieur gauche du volet [Dispositif].

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

Quand une liaison est établie, le nom du dispositif portable apparaît sur le volet [Dispositif] de rekordbox (Mac/Windows).

Remarques

Un seul dispositif portable peut être relié à rekordbox (Mac/Windows) sur l’ordinateur à la fois.

Le nom du dispositif portable apparaissant dans le volet [Dispositif] peut être vérifié et changé à l’aide de rekordbox (iOS/Android).

  • Préférences

Connexion à l’ordinateur via un câble USB

Les dispositifs Android qui prennent en charge le mode MSC agissent comme des périphériques de stockage USB (périphériques de stockage secondaires tels les clés USB) lorsqu’ils sont raccordés à l’ordinateur par un câble USB.

Raccordez le dispositif Android mis en mode MSC à l’ordinateur avec un câble USB, puis lancez rekordbox (Mac/Windows). Le nom du dispositif portable (dispositif Android) apparaît sur le volet [Dispositif] de rekordbox (Mac/Windows).

  • Pour les instructions sur le réglage du mode MSC, reportez-vous aux instructions du dispositif Android. Si l’ordinateur ne reconnaît pas votre dispositif Android comme périphérique de stockage USB (périphérique de stockage secondaire tel une clé USB), adressez-vous au fabricant du dispositif Android ou au magasin où vous l’avez acheté.

Annulation de la liaison avec l’ordinateur

Ouvrez le volet [Dispositif] dans rekordbox (Mac/Windows) sur l’ordinateur et cliquez sur à la droite de l’icône de dispositif pour annuler la liaison.

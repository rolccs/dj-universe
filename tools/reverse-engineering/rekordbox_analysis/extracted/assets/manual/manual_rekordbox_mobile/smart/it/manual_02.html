Cos’è rekordbox?

rekordbox™ è un programma che permette agli utenti che possiedono un’unità per DJ Pioneer DJ di gestire file audio da usare per esibizioni DJ.

  • rekordbox è utilizzabile per rilevare, misurare e regolare la posizione delle battute, i tempi (BPM), ecc. di file musicali.

  • rekordbox è utilizzabile per classificare e ricercare file audio e creare playlist, ecc. adatti a varie situazioni DJ.

  • rekordbox è anche utilizzabile per memorizzare informazioni su punti (per le funzioni di cue, loop , Hot Cue, ecc.) prima della performance.

Preparativi per l’uso di un dispositivo mobile

I file audio analizzati con rekordbox (Mac/Windows) di un computer, le relative playlist ecc. possono venire trasferite in rekordbox (iOS/Android) di un dispositivo mobile. Inoltre, le informazioni aggiornate con rekordbox (iOS/Android) possono venire rimandate a rekordbox (Mac/Windows) di un computer.

Collegamento ad unità DJ per performance DJ

Le unità DJ Pioneer DJ che supportano rekordbox (iOS/Android) (CDJ-2000NXS/CDJ-900NXS/XDJ-1000/XDJ-AERO/XDJ-RX) sono collegabili per le performance DJ. I file audio su dispositivi mobili possono venire esplorati comodamente dal pannello a sfioramento e caricati su lettori DJ o deck.

Feedback della storia di riproduzione

La storia dei file audio caricati su lettori DJ e riprodotti può venire memorizzata come playlist utilizzabile in futuro per performance DJ.

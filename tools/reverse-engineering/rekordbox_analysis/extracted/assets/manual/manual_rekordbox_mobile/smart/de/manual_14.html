Senden von aktualisierter Information zu einem Computer

Für Musikdateien und Playlisten, die von rekordbox (Mac/Windows) auf dem Computer zu rekordbox (iOS/Android) importiert wurden, kann die mit rekordbox (iOS/Android) aktualisierte Software in rekordbox (Mac/Windows) auf diesem Computer aktualisiert werden.

Stellen Sie eine Verknüpfung zwischen dem mobilen Gerät und dem Computer her, bevor die mit rekordbox (iOS/Android) aktualisierte Information in rekordbox (Mac/Windows) auf dem Computer reflektiert wird.

  • Verknüpfung mit einem Computer

1Öffnen Sie das [Gerät]-Bildfenster in rekordbox (Mac/Windows) im Computer.

  • Angaben zur Bedienung des Computers siehe rekordbox (Mac/Windows) Bedienungsanleitung.

2Rechtsklicken Sie auf das Icon des mobilen Geräts, das oben im [Gerät]-Bildfenster angezeigt wird.

  • Angaben zur Bedienung des Computers siehe rekordbox (Mac/Windows) Bedienungsanleitung.

  • Der Name des mobilen Geräts, das im [Gerät]-Bildfenster angezeigt wird, kann mit rekordbox (iOS/Android) geprüft und geändert werden.

    • Voreinstellungen

3Klicken Sie auf [Sammlung aktualisieren].

  • Zum Aktualisieren der Daten um einen Track zur Zeit wählen Sie die Musikdatei, rechtsklicken Sie und klicken Sie dann auf [Sammlung aktualisieren].

Die folgenden Elemente werden in der rekordbox (Mac/Windows)-Bibliothek sind aktualisiert:

  • Cue/Loop-Punkt-Speicher

  • Hot Cues

  • Beatgrid

  • Track-Information (Wertung, Farbe, Kommentare)

Hinweise

Zum Exportieren einer Playlist aus dem mobilen Gerät ziehen Sie die Playlist vom [Gerät]-Bildfenster zum [Playlist]-Bildfenster.

Zum Abbrechen der Verknüpfung mit dem Computer klicken Sie auf rechts neben dem Gerätesymbol im [Gerät]-Bildfenster in rekordbox (Mac/Windows).

Exportieren der „Meine Einstellungen“-Werte

Es ist auch möglich, die „Meine Einstellungen“-Werte, die in rekordbox (iOS/Android) gespeichert wurden, zu rekordbox (Mac/Windows) im Computer zu exportieren.

1Öffnen Sie das [Gerät]-Bildfenster in rekordbox (Mac/Windows) im Computer.

  • Angaben zur Bedienung des Computers siehe rekordbox (Mac/Windows) Bedienungsanleitung.

2Rechtsklicken Sie auf das Icon des mobilen Geräts, das im [Gerät]-Bildfenster angezeigt wird.

  • Angaben zur Bedienung des Computers siehe rekordbox (Mac/Windows) Bedienungsanleitung.

  • Der Name des mobilen Geräts, das im [Gerät]-Bildfenster angezeigt wird, kann mit rekordbox (iOS/Android) geprüft und geändert werden.

    • Voreinstellungen

3Klicken Sie auf [“Meine Einstellungen” importieren].

  • Die „Meine Einstellungen“-Werte, die in rekordbox (iOS/Android) gespeichert wurden, werden in rekordbox (Mac/Windows) auf diesem Computer reflektiert.

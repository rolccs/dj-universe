Preparation Work

Cues

When a cue is set after turning the quantize function on, the point is set to the nearest beat position on the beat grid. (Beat Snap)

1Play the track on the player panel.

2Tap [] to pause playback at the position at which you want to set the cue point.

  • The enlarged waveform display can be enlarged or reduced by pinching it in or out. The paused position can be moved in units of milliseconds by enlarging then dragging.

3When [CUE] is tapped, the position at which playback is paused is set as the cue point.

  • When [] is tapped, playback starts from the cue point.

Notes

When a new cue point is set, the previously set cue point is deleted. Save cue points you want to keep.

  • Saving cues (Cue/Loop Point Memory)

When a different music file is selected, cue points that have not been saved are canceled.

Cueing at a set cue (Back Cue/Cue Point Sampler)

Back Cue

When [CUE] is tapped, the track is cued at the set cue point and the pause mode is set.

  • When [] is tapped, playback starts from the cue point.

Cue Point Sampler

When you touch and hold [CUE], playback starts from the set cue point, and when [CUE] is released, the track returns to the cue point, cues there, and the pause mode is set.

  • When [] is tapped, playback starts from the cue point.

Setting cue points without pausing playback (Real-time Cue)

1Play the track on the player panel.

2When [IN] is tapped, the currently playing position is set as the cue point.

When [CUE] is tapped, the track is cued at the set cue point, and the pause mode is set. (Back Cue)

  • When [] is tapped, playback starts from the cue point.

When [CUE] is touched and held, playback starts from the set cue point, and when [CUE] is released, the track returns to the cue point, cues there, and the pause mode is set. (Cue Point Sampler)

  • When [] is tapped, playback starts from the cue point.

Saving cues (Cue/Loop Point Memory)

A total of up to 10 cue and loop points can be saved per music file.

Saving set cues

When [MEMORY] is tapped after setting a cue, the cue point is saved and indicated [] at the top of the waveform display.

Calling out saved cues

When [CALL ] or [CALL ] is tapped, the track is cued at the saved cue point and the pause mode is set.

  • Tap [CALL ] to call out a point before the currently playing position, [CALL ] to call out a point after the currently playing position.

  • When [] is tapped, playback starts from the cue point that was called out.

  • When [DEL] is tapped, the cue point that was called out is deleted.

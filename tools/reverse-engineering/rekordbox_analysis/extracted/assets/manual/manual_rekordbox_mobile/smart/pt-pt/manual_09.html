Ligar a um computador

O dispositivo móvel pode ser ligado através de uma LAN wireless a um computador no qual o rekordbox (Mac/Windows) está instalado para transferir dados de música entre ele e o rekordbox (Mac/Windows).

Dependendo das definições do software de segurança, o sistema operativo, etc., a ser utilizado no computador, as comunicações entre o computador e o dispositivo móvel pode ser interrompido. Para obter informações sobre o ambiente de comunicações necessário no computador (programas, sistema operativo, rede, etc.), consulte as instruções de funcionamento do rekordbox (Mac/Windows).

1Ligue o computador no qual o rekordbox (Mac/Windows) é instalado à rede LAN wireless.

  • Para obter instruções sobre a criação de ligações LAN wireless, consulte as instruções do computador ou dispositivo LAN wireless.

2Ligue o dispositivo móvel no qual o rekordbox (iOS/Android) é instalado à rede LAN wireless.

  • Para obter instruções sobre a criação de ligações LAN wireless, consulte as instruções do dispositivo móvel e dispositivo LAN wireless.

3Execute o rekordbox (Mac/Windows) no computador.

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

4No menu superior do rekordbox (iOS/Android), toque em [Connect].

rekordbox (iOS/Android) representa a operação de ligação (procura de dispositivos móveis) e efectuar no rekordbox (Mac/Windows) no computador.

5Abra o painel [Dispositivo] no rekordbox (Mac/Windows) no computador e clique em na parte inferior esquerda do painel [Dispositivo].

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

Quando a ligação estiver estabelecida, o nome do dispositivo móvel é exibido no painel rekordbox (Mac/Windows) [Dispositivo].

Notas

Apenas um dispositivo móvel pode ser ligado ao rekordbox (Mac/Windows) de cada vez no computador.

O nome do dispositivo móvel exibido no painel [Dispositivo] pode ser verificado e alterado utilizando o rekordbox (iOS/Android).

Ligar ao computador com um cabo USB

Os dispositivos Android que suportam o modo MSC funcionam como dispositivos de armazenamento USB (dispositivos de armazenamento secundário, tais como unidades de memória flash USB) quando ligados ao computador por cabo USB.

Ligue o dispositivo Android colocado no modo MSC ao computador utilizando um cabo USB e, em seguida, execute o rekordbox (Mac/Windows). O nome do dispositivo móvel (dispositivo Android) é exibido no painel [Dispositivo] no rekordbox (Mac/Windows).

  • Para obter instruções sobre como definir o modo MSC, consulte as instruções do seu dispositivo Android. Se o computador não reconhecer o seu dispositivo Android como um dispositivo de armazenamento USB (um armazenamento secundário, tal como uma unidade de memória flash USB), verifique junto do fabricante do seu dispositivo Android ou da loja onde o adquiriu.

Cancelar a ligação com o computador

Abra o painel [Dispositivo] no rekordbox (Mac/Windows) no computador e clique em na parte direita do ícone do dispositivo para cancelar a ligação.

Uso di file audio in un dispositivo mobile

Per fare uso di file audio con un dispositivo mobile, tali file devono prima venire analizzati ed aggiunti alla libreria di rekordbox (iOS/Android).

1Se [Import] viene toccato nel menu principale, appare la lista dei file audio nel dispositivo mobile.

2Toccare il o i file audio da aggiungere alla libreria. (Si possono scegliere più file.)

Il file audio scelto viene visualizzato in blu chiaro.

  • Per aggiungere i file audio visualizzati, toccare [Aggiungi tutti i brani].

3L’analisi dei file musicali inizia quando [Importa] o [Fine] viene toccato.

Finita l’analisi dei file audio, i brani vengono aggiunti alla libreria di rekordbox (iOS/Android).

Nota

I file audio del dispositivo mobile vengono simultaneamente importati (copiati) nella libreria di rekordbox (iOS/Android) quando vengono analizzati ed aggiunti alla tale libreria.

  • Se si usa un dispositivo Android, si può determinare se creare o meno copie dei file audio nella libreria di rekordbox (iOS/Android). Evitando di copiare file audio si può risparmiare spazio.

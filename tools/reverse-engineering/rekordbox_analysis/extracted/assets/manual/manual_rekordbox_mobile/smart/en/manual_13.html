Using Music Files on a Mobile Device

To use music files on the mobile device, the music files on the mobile device must first be analyzed and added to the rekordbox (iOS/Android) library.

1When [Import] is tapped at the top menu, a list of the music files on the mobile device is displayed.

2Tap the music file(s) you want to add to the library. (Multiple files can be selected.)

The selected music file is displayed in light blue.

  • To add all the music files that are displayed, tap [Add All Tracks].

3Analysis of the music file(s) begins when [Import] or [Done] is tapped.

Once analysis of the music files has finished, the tracks are added to the rekordbox (iOS/Android) library.

Note

The music files on the mobile device are simultaneously imported (copied) into the rekordbox (iOS/Android) library when they are analyzed and added to the rekordbox (iOS/Android) library.

  • When using an Android device, you can set whether or not to create copies of the music files in the rekordbox (iOS/Android) library. Storage space can be saved by not copying the music files.

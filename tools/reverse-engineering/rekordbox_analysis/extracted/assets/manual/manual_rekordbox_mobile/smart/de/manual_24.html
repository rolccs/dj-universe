DJ-Auftritte

Verwendung des Spielverlaufs

Der Spielverlauf von Musikdateien, die mindestens 1 Minute lang auf Pioneer DJ-Playern gespielt werden, wird in der Kategorie 【HISTORY】 gespeichert. Playlisten können basierend auf dem gespeicherten Spielverlauf angelegt werden.

1Tippen Sie auf dem Browser-Feld auf die 【HISTORY】-Kategorie.

Eine Liste der Spielverläufe wird angezeigt.

2Wenn Sie den Spielverlauf, den Sie als eine Playlist speichern möchten, berührt halten, wird das Aktionsblatt geöffnet.

3Wenn [Als Wiedergabeliste speichern] angetippt wird, wird das Eingabefeld für den Namen der Playlist geöffnet.

4Wenn der Name der Playlist eingegeben und dann [OK] angetippt wird, wird die neu erstellte Playlist zur 【PLAYLIST】-Kategorie hinzugefügt.

  • Bearbeiten von vorhandenen Playlisten

Löschen ungewünschter Spielverläufe

1Wenn Sie den zu löschen gewünschten Spielverlauf berührt halten, wird das Aktionsblatt geöffnet.

2Wenn [Von löschen entfernen] angetippt wird, wird der gewählte Spielverlauf gelöscht.

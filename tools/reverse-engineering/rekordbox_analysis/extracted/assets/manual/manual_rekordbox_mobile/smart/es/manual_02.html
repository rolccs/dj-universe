¿Qué es el rekordbox?

El rekordbox™ es un software que permite a los usuarios que han adquirido una unidad DJ de Pioneer DJ administrar los archivos de música que usan en sus actuaciones de DJ.

  • El rekordbox se puede usar para detectar, medir y ajustar la posiciones de tiempos de compás, tempos (BPM), etc., de archivos de música.

  • El rekordbox se puede usar para clasificar y buscar archivos de música y crear listas de reproducción, etc., para diferentes situaciones de DJ.

  • El rekordbox se puede usar para establecer y guardar información detallada de puntos (para hacer cueing, bucles, hot cueing, etc.) antes de una actuación.

Preparativos usando un aparato portátil

Los archivos de música que han sido analizados con el rekordbox (Mac/Windows) en un ordenador, las listas de reproducción, etc., se pueden cargar en un rekordbox (iOS/Android) o en un aparato portátil. Además, la información que ha sido actualizada usando el rekordbox (iOS/Android) se puede devolver al rekordbox (Mac/Windows) en un ordenador.

Conexión a unidades DJ para actuaciones de DJ

Las unidades DJ de Pioneer DJ compatibles con el rekordbox (iOS/Android) (CDJ-2000NXS/CDJ-900NXS/XDJ-1000/XDJ-AERO/XDJ-RX) se pueden conectar para actuaciones de DJ. Los archivos de música de aparatos portátiles se pueden explorar fácilmente con el panel táctil y se pueden cargar en el reproductor DJ o deck.

Retroalimentación del historial de reproducción

El historial de archivos de música que ha sido cargado en reproductores DJ y reproducido se puede guardar como listas que se pueden usar para actuaciones futuras de DJ.

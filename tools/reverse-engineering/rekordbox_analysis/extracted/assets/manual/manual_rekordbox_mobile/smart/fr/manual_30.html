Environnement réseau

Selon les réglages de l’antivirus, le système d’exploitation, etc. utilisés sur l’ordinateur, la communication entre l’ordinateur et le dispositif portable peut être interrompue.

  • Pour le détail sur l’environnement requis sur l’ordinateur pour la communication (programmes, système d’exploitation, réseau, etc.), reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

En outre, si le réseau (adresse IP, numéro de port, etc.) est restreint par le dispositif de communication (routeur, etc.), la communication avec les appareils DJ ou les ordinateurs peut être interrompue.

  • Pour les instructions sur les dispositifs de communication utilisés, contactez les fabricants ou les revendeurs.

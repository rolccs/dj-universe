Pannello del browser

Usato per tornare al livello superiore.

La locazione (livello) attualmente esplorata viene visualizzata qui.

  • Esplorazione di file audio

Se si è collegati a lettori DJ, viene visualizzato il loro stato di riproduzione.

Viene visualizzata qui una lista degli album, titoli, ecc. nella locazione (livello) attualmente esplorata. Toccando un brano, il pannello del lettore si apre e la riproduzione inizia.

Se un brano viene sfiorato mentre si è collegati a lettori DJ, il pannello di collegamento (CDJ/XDJ) o il pannello di collegamento (XDJ) si apre.

Usato per scorrere la lista fino alla riga che inizia dalla lettera specificata.

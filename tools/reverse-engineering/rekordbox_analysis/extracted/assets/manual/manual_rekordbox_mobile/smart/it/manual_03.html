Per i nuovi utenti

Prima di tutto, preparare i brani da usare per la performance DJ. Analizzare i file audio del dispositivo mobile ed aggiungerli alla libreria di rekordbox (iOS/Android).

  • Uso di file audio in un dispositivo mobile

Se l’uso viene fatto con rekordbox (Mac/Windows) già installato nel computer, i file già analizzati con rekordbox (Mac/Windows), le playlist, ecc., possono venire importati in rekordbox (iOS/Android).

Enlace con una unidad XDJ

Modelos aplicables: XDJ-AERO

Los aparatos portátiles se pueden conectar mediante LAN inalámbrica a unidades DJ de Pioneer DJ compatibles con el rekordbox (iOS/Android) para intercambiar datos de música.

1Conecte a la red usando la función de LAN inalámbrica interna de la unidad DJ.

  • Para conocer detalles de los tipos y del procedimiento de conexión de LAN inalámbrica, consulte los manuales de instrucciones de las unidades DJ respectivas.

  • XDJ-AERO Guía de conexión

2Conecte el aparato portátil en el que está instalado el rekordbox (iOS/Android) a la red LAN inalámbrica.

  • Para conocer instrucciones de cómo hacer las conexiones de LAN inalámbrica, consulte las instrucciones de su aparato portátil.

3En el menú inicial del rekordbox (iOS/Android), toque [Load].

Cuando se encuentra una unidad DJ con la que es posible comunicarse, el rekordbox (iOS/Android) intenta automáticamente conectarse con ella.

Cuando se establece el enlace, el panel explorador se abre.

Notas

Con la unidad DJ se pueden enlazar simultáneamente hasta cuatro aparatos portátiles.

El nombre del aparato portátil visualizado en la unidad DJ se puede verificar y cambiar usando el rekordbox (iOS/Android).

  • Preferencias

Los aparatos Android establecidos en el modo MSC funcionan como aparatos de almacenamiento USB (aparatos de almacenamiento secundarios tales como unidades de memoria flash USB) cuando se conectan a una unidad DJ con un cable USB. En este caso, los datos no se pueden transferir usando el rekordbox (iOS/Android). Para detalles de las conexiones entre unidades DJ y aparatos de almacenamiento USB, consulte el manual de instrucciones de la unidad DJ respectiva.

Cancelación del enlace con la unidad DJ

En el panel explorador, cuando toca o [Menu] en la barra de herramientas y luego vuelve al menú inicial, el enlace con la unidad DJ se cancela.

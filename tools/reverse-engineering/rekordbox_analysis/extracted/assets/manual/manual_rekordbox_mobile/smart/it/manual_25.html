Impostazioni

Preferenze

Se [Settings] del menu principale o pannello del browser viene toccato, viene visualizzata la lista delle voci da impostare.

Lista delle voci di impostazione

Nome del dispositivo

Digitare il nome di questo dispositivo mobile come deve venire visualizzato dalle unità DJ (CDJ-2000NXS/CDJ-900NXS/XDJ-1000/XDJ-AERO/XDJ-RX) ed in rekordbox (Mac/Windows) del computer.

Blocco del caricamento

Questo determina se permettere o meno il caricamento di nuovi file audio quando il lettore DJ su cui si sta tentando di caricare file musicali sta riproducendo.

Modalità di analisi di brani

Determina la modalità di analisi dei file audio. Scegliere [Normale] per i file audio di tempo costante, [Dinamico] per quelli il cui tempo varia.

Rilevamento della tonalità

Determina se rilevare simultaneamente la chiave (altezza) nell’analizzare un file audio.

Modalità di visualizzazione dei tempi

Specifica la modalità di visualizzazione (Tempo trascorso/Tempo rimanente) per il tempo visualizzato sul pannello del lettore di rekordbox (iOS/Android).

Metronomo

Specifica il suono di un metronomo che suona durante la regolazione della beatgrid di file audio.

  • Opzioni di regolazione della beatgrid

Categorie attive

Specifica le voci da visualizzare nel menu di categoria nel pannello del browser di rekordbox (iOS/Android) e nel display dell’unità principale del lettore DJ.

  • Impostazione delle categorie attive e delle opzioni di riordino attive

Opzioni attive

Questo specifica le voci da visualizzare nel menu di riordino del display principale del lettore DJ.

  • Impostazione delle categorie attive e delle opzioni di riordino attive

Categoria scelta dall’utente

Specifica la categoria scelta dall’utente da visualizzare subito a destra dei titoli durante l’esplorazione di file audio nel display dell’unità principale o di un lettore DJ (ad esempio un CDJ-2000NXS).

  • Per quanto riguarda l’esplorazione di librerie con lettori DJ, consultare le istruzioni per l’uso del rispettivo lettore DJ.

Colore

Questo fornisce anche descrizioni del codice a colori (otto colori) usato per esplorare file audio sul display dell’unità principale di un lettore DJ.

  • Per quanto riguarda l’esplorazione di librerie con lettori DJ, consultare le istruzioni per l’uso del rispettivo lettore DJ.

Disposizione delle icone dei CDJ/XDJ

L’ordine delle icone del lettore DJ visualizzate nel pannello dei link (CDJ/XDJ) può essere controllato e modificato a rispecchiare l’ordine effettivo del proprio lettore DJ.

  • Performance con una unità CDJ/XDJ propria

My Settings

Le impostazione dell’unità principale del lettore DJ possono essere salvate in rekordbox (iOS/Android) e le stesse impostazioni possono venire applicate ai rispettivi lettori DJ. Questo permette agli utenti di impostare immediatamente i lettori DJ con le impostazioni cui sono abituati, per poter dedicare tutta la propria attenzione alla performance.

  • My Settings

Caricamento automatico degli Hot Cue nel CDJ/XDJ

Nello scegliere file musicali e caricandoli su lettori DJ, gli hot cue (A, B e C) del lettore DJ (ad es. CDJ-2000NXS) possono essere sostituiti a forza con gli hot cue memorizzati nei file audio caricati.

  • Performance con una unità CDJ/XDJ propria

Posizione della libreria (solo dispositivi Android)

Specifica la locazione di salvataggio della libreria di rekordbox (iOS/Android) (un database contenente informazioni su file audio, playlist, ecc.).

Se la libreria di rekordbox (iOS/Android) è salvata nella memoria del dispositivo Android, è possibile guadagnare spazio trasferendola in una scheda SD.

Attenzione

Trasferendo la libreria i suoi file audio precedentemente importati non sono più raggiungibili e vanno reimportati.

Copia i brani al momento dell'importazione (solo dispositivi Android)

Determina se i file audio analizzati da dispositivi Android vanno copiati nella libreria di rekordbox (iOS/Android) ed aggiunti ad essa. Tenere presente che se il percorso (intero) del file audio è lungo oltre 255 caratteri, una copia del file viene sempre fatta nella libreria di rekordbox (iOS/Android).

  • Uso di file audio in un dispositivo mobile

Elimina i file temporanei

Se la connessione viene interrotta o il programma viene interrotto durante l’importazione di file audio o playlist da rekordbox (Mac/Windows), dei file temporanei finiranno col rimanere nella libreria di rekordbox (iOS/Android) (un database di informazioni sui file audio, playlist, ecc.). Questa funzione consente di trovare e cancellare file temporanei non più necessari, liberando spazio.

Impostazione delle categorie attive e delle opzioni di riordino attive

Aggiunta e cancellazione di categorie attive

1Se [Categorie attive] viene toccato, appare una lista delle categorie attive già impostate.

2Toccare prima [Modifica] e poi .

Appare una lista delle categorie non visualizzate.

3Toccare la o le categorie da visualizzare. (Si possono aggiungere più categorie.)

4Toccare [Fine] per chiudere la lista delle categorie non visualizzate.

5Trascinare verso il lato destro della categoria per cambiare l’ordine di visualizzazione delle categorie attive.

Si può far sì che una categoria attiva non venga più visualizzata toccando sulla sinistra della categoria e poi [Disattivare].

6Toccare [Fine] per finire di impostare le categorie attive.

Aggiunta e cancellazione di opzioni di riordino attive

The opzioni di riordino attive possono anche venire impostate con lo stesso tipo di procedura descritto in Aggiunta e cancellazione di categorie attive.

Подготовка

Для клиентов, пользующихся впервые

Прежде всего подготовьте дорожки, которые будут использоваться для DJ исполнения. Проанализируйте музыкальные файлы на мобильном устройстве и добавьте их в библиотеку rekordbox (iOS/Android).

  • Использование музыкальных файлов на мобильном устройстве

При использовании с уже установленным на компьютере rekordbox (Mac/Windows), музыкальные файлы, уже проанализированные с помощью rekordbox (Mac/Windows), списки воспроизведения, др., могут быть импортированы в rekordbox (iOS/Android).

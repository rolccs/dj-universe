DJ-Auftritte

Auswählen von Musik für eine Taglist

Die Tag List ist eine Liste, die in Echtzeit zwischen mehreren über ein LAN verbundenen DJ-Playern geteilt werden kann.

1Wenn ein Track im Browser-Feld berührt gehalten wird, wird das Aktionsblatt geöffnet.

2Wenn [Zur Tag-Liste hinzufügen] angetippt wird, wird der gewählte Track zur 【TAGLIST】-Kategorie hinzugefügt.

  • Für Anweisungen zum Zugriff auf die Taglisten von DJ-Playern siehe die Bedienungsanleitung für die betreffenden DJ-Player.

Löschen von Tracks aus Taglisten

Zum Löschen einzelner Tracks

1Tippen Sie auf dem Browser-Feld auf die 【TAGLIST】-Kategorie.

Eine Liste der Tracks, die zur Taglist hinzugefügt wurden, wird angezeigt.

2Wenn Sie den zu löschen gewünschten Track berührt halten, wird das Aktionsblatt geöffnet.

3Wenn [Von Tag-Liste entfernen] angetippt wird, wird der gewählte Track von der 【TAGLIST】-Kategorie gelöscht.

Zum Löschen aller Tracks

1Wenn die 【TAGLIST】-Kategorie im Browser-Feld berührt gehalten wird, wird das Aktionsblatt geöffnet.

2Wenn [Verlauf Tag-Liste] angetippt wird, werden alle Tracks von der 【TAGLIST】-Kategorie gelöscht.

Speichern Sie Taglisten als Playlisten

Es ist auch möglich, Taglisten, die für Auftritte verwendet wurden, als Playlisten zu speichern.

1Wenn die 【TAGLIST】-Kategorie im Browser-Feld berührt gehalten wird, wird das Aktionsblatt geöffnet.

2Wenn [Als Wiedergabeliste speichern] angetippt wird, wird das Eingabefeld für den Namen der Playlist geöffnet.

3Wenn der Name der Playlist eingegeben und dann [OK] angetippt wird, wird die neu erstellte Playlist zur 【PLAYLIST】-Kategorie hinzugefügt.

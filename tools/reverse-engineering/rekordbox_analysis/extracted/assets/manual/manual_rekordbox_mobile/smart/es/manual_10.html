Enlace con unidades CDJ/XDJ

Modelos aplicables: CDJ-2000NXS/CDJ-900NXS/XDJ-1000/XDJ-RX

Los aparatos portátiles se pueden conectar a reproductores DJ de Pioneer DJ compatibles con el rekordbox (iOS/Android) mediante un aparato de LAN inalámbrica para intercambiar datos de música.

1Conecte el reproductor DJ a una red con punto de acceso de LAN inalámbrica (PRO DJ LINK).

  • Para conocer detalles de los tipos y del procedimiento de conexión de LAN inalámbrica, consulte los manuales de instrucciones de los reproductores DJ respectivos y de las instrucciones de los aparatos de LAN inalámbrica.

  • CDJ-2000NXS Guía de conexión

2Conecte el aparato portátil en el que está instalado el rekordbox (iOS/Android) a la red LAN inalámbrica.

  • Para conocer instrucciones de cómo hacer las conexiones de LAN inalámbrica, consulte las instrucciones de su aparato portátil y del aparato de LAN inalámbrica.

3En el menú inicial del rekordbox (iOS/Android), toque [Load].

Cuando se encuentra un reproductor DJ con el que se pueden establecer comunicaciones, el rekordbox (iOS/Android) se pone en espera para la operación de conexión (autorización de enlace) que va a hacerse en el reproductor DJ.

4En el reproductor DJ, especifique el aparato portátil cuyo enlace va a ser autorizado.

  • Para conocer la operación de autorización del enlace, consulte los manuales de instrucciones de los reproductores DJ respectivos.

  • CDJ-2000NXS Guía de conexión

Cuando se establece el enlace, el panel explorador se abre.

Notas

Con el reproductor DJ se pueden enlazar simultáneamente hasta cuatro aparatos portátiles. (* Para XDJ-RX, hasta dos dispositivos móviles se pueden vincular.)

El nombre del aparato portátil visualizado en el reproductor DJ se puede verificar y cambiar usando el rekordbox (iOS/Android).

  • Preferencias

Conexión a un reproductor DJ con un cable USB

Los aparatos iOS (tales como los iPhone) y los aparatos Android compatibles con el modo MTP (Android 3.0 y posteriores) se pueden conectar al reproductor DJ usando un cable USB.

Después de conectar el aparato portátil y el reproductor DJ con un cable USB, toque [Load] en el menú inicial del rekordbox (iOS/Android). Una vez establecido el enlace, el panel explorador se abre.

  • Para instrucciones sobre cómo establecer el modo MTP, restricciones en las operaciones desde el aparato Android, restricciones de comunicación, etc., consulte las instrucciones de su aparato Android.

  • Los aparatos Android establecidos en el modo MSC funcionan como aparatos de almacenamiento USB (aparatos de almacenamiento secundarios tales como unidades de memoria flash USB) cuando se conectan a un reproductor DJ con un cable USB. En este caso, los datos no se pueden transferir usando el rekordbox (iOS/Android). Para detalles de las conexiones entre reproductores DJ y aparatos de almacenamiento USB, consulte el manual de instrucciones del reproductor DJ respectivo.

Cancelación del enlace con el reproductor DJ

En el panel explorador, cuando toca o [Menu] en la barra de herramientas y luego vuelve al menú inicial, el enlace con el reproductor DJ se cancela.

各画面でできること

トップメニュー

本ソフトウェアが起動するとトップメニューが表示されます。

Prepare

各種ポイント情報やプレイリストなどを作成して、 DJプレイの前準備をします。

  • ブラウザーパネル

  • プレーヤーパネル

Load

DJプレーヤーと接続してDJプレイを行います。

  • DJ機器 (CDJ/XDJ) に接続する

  • DJ機器 (XDJ) に接続する

Connect

コンピューター上のrekordbox (Mac/Windows用) ですでに解析済みの音楽ファイルおよび各種ポイント情報やプレイリストなどを本ソフトウェアへ取り込みます。 また、 本ソフトウェアで更新した各種ポイント情報やプレイリストなどをrekordbox (Mac/Windows用) へ反映させます。

  • コンピューターから取り込む

  • コンピューターへ反映させる

Import

モバイルデバイス内の音楽ファイルを解析して本ソフトウェアのライブラリに追加します。

  • モバイルデバイス内の音楽ファイルを使う

Help

ヘルプメニューを開きます。

Settings

各種の設定を確認・変更します。

  • 本ソフトウェアの設定

ツールバー

画面の下部に表示されるツールバーのアイコンについては、 下記の表をご覧ください。

アイコン

説明

Menu

トップメニューに戻ります。

Browse

ブラウザーパネルの最上位の階層に戻ります。

Playing

プレーヤーパネルが開いて、 再生中の曲が表示されます。

USB

DJ機器 (XDJ-AERO) にセットされたUSBデバイス内のコンテンツをブラウズして、 DJ機器 (XDJ-AERO) で再生します。

  • USBデバイスの取り扱いについては、 各DJ機器の取扱説明書をご覧ください。

Import

モバイルデバイス内の音楽ファイルを解析して本ソフトウェアのライブラリに追加します。

  • モバイルデバイス内の音楽ファイルを使う

Settings

各種の設定を確認・変更します。

  • 本ソフトウェアの設定

Help

ヘルプ (ユーザーマニュアル) を開きます。

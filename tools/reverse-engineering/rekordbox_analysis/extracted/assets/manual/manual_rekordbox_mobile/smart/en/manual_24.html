Using the Playing History

The playing history of music files played for at least 1 minute on Pioneer DJ players is stored in the 【HISTORY】 category. Playlists can be created based on the stored playing history.

1On the browser panel, tap the 【HISTORY】 category.

A list of playing histories is displayed.

2When the playing history you want to store as a playlist is touched and held, the action sheet opens.

3When [Save as Playlist] is tapped, the playlist name input box opens.

4When the playlist name is input then [OK] is tapped, the newly created playlist is added to the 【PLAYLIST】 category.

  • Editing existing playlists

Deleting unnecessary playing histories

1When the playing history you want to delete is touched and held, the action sheet opens.

2When [Delete History] is tapped, the selected playing history is deleted.

Einstellungen

Meine Einstellungen

Zutreffende Modelle: CDJ-2000NXS/CDJ-900NXS/XDJ-1000

Die Einstellungen einer DJ-Player-Haupteinheit können in rekordbox (iOS/Android) gespeichert werden, und die gleichen Einstellungen können für die jeweiligen DJ-Player übernommen werden. Dies erlaubt es Benutzern, sofort DJ-Player mit den Einstellungen einzurichten, an die sie gewöhnt sind, so dass sie ihre volle Aufmerksamkeit dem aktuellen DJ-Auftritt widmen können.

1Wenn [Settings] im Top-Menü oder Browser-Feld angetippt wird, wird die Liste der Einstellelemente angezeigt.

2Wenn [Meine Einstellungen] angetippt wird, werden die bei „Meine Einstellungen“ einstellbaren Elemente angezeigt.

3Tippen Sie auf die Einstelloption, und ändern Sie den Einstellwert.

  • Für Elemente, die bei „Meine Einstellungen“ eingestellt werden können, siehe Bedienungsanleitung des betreffenden DJ-Players.

4Reflektieren Sie die Werte von „Meine Einstellungen“ in den betreffenden DJ-Playern.

  • Reflektieren der Werte von „Meine Einstellungen“ in den DJ-Einheiten

Hinweise

Es ist auch möglich, die „Meine Einstellungen“-Werte, die in rekordbox (Mac/Windows) in dem Computer gespeichert wurden, in rekordbox (iOS/Android) zu importieren.

  • Importieren von einem Computer

Es ist auch möglich, die „Meine Einstellungen“-Werte, die in rekordbox (iOS/Android) gespeichert wurden, zu rekordbox (Mac/Windows) zu exportieren.

  • Senden von aktualisierter Information zu einem Computer

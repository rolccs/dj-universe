Verwendung von Musikdateien auf einem mobilen Gerät

Zur Verwendung von Musikdateien auf dem mobilen Gerät müssen die Musikdateien auf dem mobilen Gerät zuerst analysiert und zur rekordbox (iOS/Android)-Bibliothek hinzugefügt werden.

1Wenn [Import] im Top-Menü angetippt wird, wird eine Liste der Musikdateien auf dem mobilen Gerät angezeigt.

2Tippen Sie die Musikdatei(en) an, die Sie zur Bibliothek hinzufügen wollen. (Mehrere Dateien können gewählt werden.)

Die gewählte Musikdatei wird in Hellblau angezeigt.

  • Zum Hinzufügen aller angezeigten Musikdateien tippen Sie auf [Alle Tracks hinzufügen].

3Die Analyse der Musikdatei(en) beginnt, wenn [Importieren] oder [Fertig] angetippt wird.

Wenn die Analyse der Musikdateien fertig ist, werden die Tracks zur rekordbox (iOS/Android)-Bibliothek hinzugefügt.

Hinweis

Die Musikdateien im mobilen Gerät werden simultan in die Bibliothek von rekordbox (iOS/Android) importiert (kopiert), wenn sie analysiert werden, und werden zur Bibliothek von rekordbox (iOS/Android) hinzugefügt.

  • Bei Verwendung eines Android-Geräts können Sie wählen, ob Sie Kopien der Musikdateien in der Bibliothek von rekordbox (iOS/Android) erstellen. Speicherplatz kann gespart werden, indem die Musikdateien nicht kopiert werden.

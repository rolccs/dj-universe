Connexions

Liaison avec des appareils CDJ/XDJ

Modèles applicables : CDJ-2000NXS/CDJ-900NXS/XDJ-1000/XDJ-RX

Des dispositifs portables peuvent se connecter aux lecteurs DJ Pioneer DJ prenant en charge rekordbox (iOS/Android) via un dispositif sans fil pour échanger des données musicales.

1Raccordez le lecteur DJ à un réseau avec un point d’accès LAN sans fil (PRO DJ LINK).

  • Pour le détail sur les types de connexions sans fil et la marche à suivre pour se connecter, reportez-vous aux modes d’emploi du lecteur DJ et du dispositif sans fil.

  • CDJ-2000NXS Guide de raccordement

2Connectez le dispositif portable sur lequel rekordbox (iOS/Android) est installé au réseau sans fil.

  • Pour les instructions sur la réalisation de connexions LAN sans fil, reportez-vous aux instructions de votre dispositif portable et du dispositif sans fil.

3Sur le menu principal de rekordbox (iOS/Android), tapez sur [Load].

Lorsqu’un lecteur DJ avec lequel une communication peut être établie est trouvé, rekordbox (iOS/Android) attend que la connexion (autorisation de liaison) soit effectuée sur le lecteur DJ.

4Sur le lecteur DJ, spécifiez le dispositif portable dont la liaison doit être autorisée.

  • Pour l’autorisation de la liaison, reportez-vous aux modes d’emploi des lecteurs DJ utilisés.

  • CDJ-2000NXS Guide de raccordement

Lorsque la liaison est établie, le panneau d’exploration s’ouvre.

Remarques

Jusqu’à quatre dispositifs portables peuvent être reliés simultanément au lecteur DJ. (* Pour XDJ-RX, jusqu'à deux appareils mobiles peuvent être liés.)

Le nom du dispositif portable apparaissant sur le lecteur DJ peut être vérifié et changé à l’aide de rekordbox (iOS/Android).

Connexion à un lecteur DJ via un câble USB

Les dispositifs iOS (comme les iPhone) et les dispositifs Android prenant en charge le mode MTP (Android 3.0 et ultérieur) peuvent être raccordés aux lecteurs DJ avec un câble USB.

Après avoir raccordé le dispositif portable et le lecteur DJ avec un câble USB, tapez sur [Load] sur le menu principal de rekordbox (iOS/Android). Lorsque la liaison est établie, le panneau d’exploration s’ouvre.

  • Pour les instructions sur le réglage du mode MTP, les restrictions concernant les opérations depuis le dispositif Android, les restrictions concernant les communications, etc. reportez-vous instructions du dispositif Android.

  • Les dispositifs Android réglés sur le mode MSC agissent comme des périphériques de stockage USB (périphériques de stockage secondaires tels les clés USB) lorsqu’ils sont raccordés à un lecteur DJ par un câble USB. Dans ce cas, les données ne peuvent pas être transférées à l’aide de rekordbox (iOS/Android). Pour plus d’informations sur les liaisons entre des lecteurs DJ et les périphériques de stockage USB, reportez-vous au mode d’emploi des lecteurs DJ.

Annulation de la liaison avec le lecteur DJ

Sur le panneau d’exploration, lorsque vous tapez sur ou sur [Menu] dans la barre d’outils puis revenez au menu principal, la liaison au lecteur DJ est annulée.

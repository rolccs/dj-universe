Importazione da un computer

Le playlist ed i file audio analizzati con rekordbox (Mac/Windows) su di un computer possono venire importati in rekordbox (iOS/Android).

Prima di importare file audio, playlist, ecc. da rekordbox (Mac/Windows) del computer, collegare il dispositivo mobile al computer.

  • Collegamento con un computer

1Aprire i riquadri [Dispositivo] e [Collezione] uno accanto all’altro in rekordbox (Mac/Windows) del computer.

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

2Trascinare il file musicale dal riquadro [Collezione] al riquadro [Dispositivo].

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

  • Il nome del dispositivo mobile visualizzato nel riquadro [Dispositivo] è controllabile e modificabile con rekordbox (iOS/Android).

    • Preferenze

I vari tipi di informazione su punti, ecc. preparati con rekordbox (Mac/Windows) vengono importati nella libreria di rekordbox (iOS/Android) insieme ai file audio.

Nota

Per cancellare il collegamento col computer, fare clic su alla destra dell’icona del dispositivo nel riquadro [Dispositivo] in rekordbox (Mac/Windows).

Importazione di playlist

Le playlist creati con rekordbox (Mac/Windows) su di un computer possono venire importati in rekordbox (iOS/Android).

1Aprire i riquadri [Dispositivo] e [Playlist] uno accanto all’altro in rekordbox (Mac/Windows) del computer.

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

2Trascinare la playlist dal riquadro [Playlist] al riquadro [Dispositivo].

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

I vari tipi di informazione su punti, ecc. preparati con rekordbox (Mac/Windows) vengono importati nella libreria di rekordbox (iOS/Android) insieme alle playlist ed ai file audio.

Importazione dei valori “My Settings”

Si possono anche importare i valori “My Settings” memorizzati in rekordbox (Mac/Windows) di un computer in rekordbox (iOS/Android).

1In rekordbox (Mac/Windows) del computer, scegliere [Preferenze] dal menu [File] per fare aprire la schermata [Preferenze].

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

2Nella schermata [Preferenze], scegliere [Esportazione] > [My Settings], poi aprire la scheda [My Settings].

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

3Nella scheda [My Settings], fare clic su [Applica a tutti i dispositivi collegati].

  • Per quanto riguarda le operazioni col computer, vedere le istruzioni per l’uso di rekordbox (Mac/Windows).

  • Se [Applica automaticamente le modifiche ai dispositivi] è spuntato, i valori “My Settings” stabiliti con rekordbox (Mac/Windows) verranno automaticamente importati in rekordbox (iOS/Android) dalla prossima volta.

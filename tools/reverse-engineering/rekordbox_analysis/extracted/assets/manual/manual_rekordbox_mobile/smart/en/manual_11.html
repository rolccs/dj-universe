Connections

Linking with an XDJ Unit

Applicable models: XDJ-AERO

Mobile devices can be connected by wireless LAN to Pioneer DJ units supporting rekordbox (iOS/Android) to exchange music data.

1Connect to the network using the DJ unit’s internal wireless LAN function.

  • For details on the wireless LAN connection types and connection procedure, see the respective DJ unit’s operating instructions.

  • XDJ-AERO Connection Guide

2Connect the mobile device on which rekordbox (iOS/Android) is installed to the wireless LAN network.

  • For instructions on making the wireless LAN connections, see your mobile device’s instructions.

3On rekordbox (iOS/Android)’s top menu, tap [Load].

When a DJ unit with which communications are possible is found, rekordbox (iOS/Android) automatically attempts to connect to it.

When the link is established, the browser panel opens.

Notes

Up to four mobile devices can be linked simultaneously to the DJ unit.

The name of the mobile device displayed on the DJ unit can be checked and changed using rekordbox (iOS/Android).

  • Preferences

Android devices set to the MSC mode operate as USB storage devices (secondary storage devices such as USB flash memory drives) when connected to DJ unit by USB cable. In this case, data cannot be transferred using rekordbox (iOS/Android). For details on connections between DJ units and USB storage devices, see the respective DJ unit’s operating instructions.

Canceling the link with the DJ unit

On the browser panel, when you tap or [Menu] on the toolbar then return to the top menu, the link to the DJ unit is canceled.

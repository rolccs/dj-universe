Entorno de red

Dependiendo de los ajustes del software de seguridad, el sistema operativo, etc., usados en el ordenador, las comunicaciones entre el ordenador y el aparato portátil puede que se interrumpan.

  • Para conocer detalles del entorno de comunicación requerido en el ordenador (programas, sistema operativo, red, etc.), consulte el manual de instrucciones del rekordbox (Mac/Windows).

Además, si la red (dirección IP, números de puertos, etc.) está restringida por un aparato de comunicación (enrutador, etc.), las comunicaciones con unidades DJ u ordenadores puede que se interrumpan.

  • Para instrucciones de cómo configurar los aparatos de comunicación que está usando, póngase en contacto con los fabricantes o agentes de ventas respectivos.

O que é o rekordbox?

O rekordbox™ é um programa de software que permite aos utilizadores que adquiriram uma unidade de DJ da Pioneer DJ gerir os ficheiros de música que utilizam em apresentações de DJ.

  • O rekordbox pode ser utilizado para detectar, medir e ajustar as posições de batida, tempos (BPMs), etc., de ficheiros de música.

  • O rekordbox pode ser utilizado para classificar e pesquisar ficheiros de música e criar listas de reprodução, etc. para diferentes aplicações de DJ.

  • O rekordbox pode ser utilizado para definir e armazenar informações detalhadas de pontos (para Cue, Loop, Hot Cue, etc.) antes de uma apresentação.

Efectuar os preparativos utilizando um dispositivo móvel

Os ficheiros de música que foram analisados com o rekordbox (Mac/Windows) num computador, listas de reprodução, etc., podem ser carregados para o rekordbox (iOS/Android) num dispositivo móvel. Além disso, as informações que foram actualizadas utilizando o rekordbox (iOS/Android) podem ser reenviadas para o rekordbox (Mac/Windows) num computador.

Ligar a unidades de DJ para actuações de DJ

As unidades de DJ da Pioneer DJ que suportam o rekordbox (iOS/Android) (CDJ-2000NXS/CDJ-900NXS/XDJ-1000/XDJ-AERO/XDJ-RX) podem ser ligadas para realizar actuações de DJ. Os ficheiros de música em dispositivos móveis podem ser explorados confortavelmente utilizando o painel táctil e carregados no leitor de DJ ou deck.

Feedback do histórico de reprodução

O histórico de ficheiros de música que foram transferidos para leitores de DJ e reproduzidos podem ser armazenados como listas de reprodução que podem ser utilizadas em futuras actuações de DJ.

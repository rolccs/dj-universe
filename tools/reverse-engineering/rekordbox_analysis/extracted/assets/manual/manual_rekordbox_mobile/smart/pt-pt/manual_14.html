Enviar informações actualizadas para um computador

Para ficheiros de música e listas de reprodução importados a partir do rekordbox (Mac/Windows) no computador para o rekordbox (iOS/Android), as informações actualizadas utilizando o rekordbox (iOS/Android) podem ser actualizadas no rekordbox (Mac/Windows) no computador.

Estabeleça uma ligação entre o dispositivo móvel e o computador antes de reflectir as informações que foram actualizadas utilizando o rekordbox (iOS/Android) no rekordbox (Mac/Windows) no computador.

  • Ligar a um computador

1Abra o painel [Dispositivo] no rekordbox (Mac/Windows) no computador.

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

2Clique com o botão direito do rato no ícone do dispositivo móvel na parte superior do painel [Dispositivo].

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

  • O nome do dispositivo móvel exibido no painel [Dispositivo] pode ser verificado e alterado utilizando o rekordbox (iOS/Android).

    • Preferências

3Clique em [Actualizar Colecção].

  • Para actualizar uma faixa de cada vez, seleccione o ficheiro de música, clique com o botão direito do rato e, em seguida, clique em [Actualizar Colecção].

Os seguintes itens são actualizados na biblioteca do rekordbox (Mac/Windows):

  • Memórias de cue/loop point

  • Hot cues

  • Grelha de batida

  • Informações das faixas (Classificação, Cor, Comentários)

Notas

Para exportar uma lista de reprodução a partir do dispositivo móvel, arraste a lista de reprodução a partir do painel [Dispositivo] para o painel [Lista de Reprodução].

Para cancelar a ligação com o computador, clique em à direita do ícone do dispositivo no painel [Dispositivo] no rekordbox (Mac/Windows).

Exportar os valores de “Minhas definições”

Também é possível exportar os valores de “Minhas definições” armazenados no rekordbox (iOS/Android) para o rekordbox (Mac/Windows) no computador.

1Abra o painel [Dispositivo] no rekordbox (Mac/Windows) no computador.

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

2Clique com o botão direito do rato no ícone do dispositivo móvel exibido no painel [Dispositivo].

  • Para obter informações sobre as operações no computador, consulte as manual de instruções do rekordbox (Mac/Windows).

  • O nome do dispositivo móvel exibido no painel [Dispositivo] pode ser verificado e alterado utilizando o rekordbox (iOS/Android).

    • Preferências

3Clique em [Importar “Minhas definições”].

  • Os valores de “Minhas definições” armazenados no rekordbox (iOS/Android) são reflectidos no rekordbox (Mac/Windows) no computador.

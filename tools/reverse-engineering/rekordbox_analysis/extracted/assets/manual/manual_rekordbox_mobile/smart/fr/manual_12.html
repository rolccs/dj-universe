Importation depuis un ordinateur

Les listes de lecture et les fichiers de musique analysés avec rekordbox (Mac/Windows) sur l’ordinateur peuvent être importés dans rekordbox (iOS/Android).

Avant d’importer des fichiers de musique, des listes de lecture, etc. de rekordbox (Mac/Windows) sur l’ordinateur, établissez la liaison entre le dispositif portable et l’ordinateur.

  • Liaison à un ordinateur

1Ouvrez les volets [Dispositif] et [Collection] l’un à côté de l’autre dans rekordbox (Mac/Windows) sur l’ordinateur.

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

2Tirez le fichier de musique du volet [Collection] pour le déposer dans le volet [Dispositif].

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

  • Le nom du dispositif portable apparaissant dans le volet [Dispositif] peut être vérifié et changé à l’aide de rekordbox (iOS/Android).

    • Préférences

Les différents types d’informations concernant les points, etc. préparés avec rekordbox (Mac/Windows) sont importés dans la bibliothèque rekordbox (iOS/Android) en même temps que les fichiers de musique.

Remarque

Pour annuler la liaison avec l’ordinateur, cliquez sur à la droite de l’icône de dispositif sur le volet [Dispositif] dans rekordbox (Mac/Windows).

Importation de listes de lecture

Les listes de lecture créées avec rekordbox (Mac/Windows) sur l’ordinateur peuvent être importées dans rekordbox (iOS/Android).

1Ouvrez les volets [Dispositif] et [Liste de lecture] l’un à côté de l’autre dans rekordbox (Mac/Windows) sur l’ordinateur.

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

2Tirez la liste de lecture du volet [Liste de lecture] pour la déposer dans le volet [Dispositif].

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

Les différents types d’informations concernant les points, etc. préparés avec rekordbox (Mac/Windows) sont importés dans la bibliothèque rekordbox (iOS/Android) en même temps que les listes de lecture et les fichiers de musique.

Importation des valeurs de “Mes réglages”

Il est également possible d’importer dans rekordbox (iOS/Android) les valeurs de “Mes réglages” enregistrées dans rekordbox (Mac/Windows) sur l’ordinateur.

1Dans rekordbox (Mac/Windows) sur l’ordinateur, sélectionnez [Préférences] dans le menu [Fichier] pour ouvrir l’écran [Préférences].

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

2Sur l’écran [Préférences], sélectionnez [Exportation] > [Mes Réglages], puis ouvrez l’onglet [Mes Réglages].

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

3Sur l’onglet [Mes Réglages], cliquez sur [Appliquer aux tous les dispositifs connectés].

  • Pour le fonctionnement de l’ordinateur, reportez-vous au mode d’emploi de rekordbox (Mac/Windows).

  • Lorsque [Appliquer automatiquement les modifications aux dispositifs] est coché, les valeurs de “Mes réglages” définies avec rekordbox (Mac/Windows) sont automatiquement importées dans rekordbox (iOS/Android) la fois suivante.

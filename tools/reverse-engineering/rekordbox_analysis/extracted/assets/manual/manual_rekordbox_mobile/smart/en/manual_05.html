Browser Panel

Used to return to the level above.

  • Browsing Music Files

When linked to DJ players, the DJ player’s playing status is displayed.

A list of the albums, titles, etc., in the location (level) currently being browsed is displayed here. When a track is tapped, the player panel opens and playback starts.

When a track is tapped while linked to DJ players, the link panel (CDJs/XDJs) or link panel (XDJ) opens.

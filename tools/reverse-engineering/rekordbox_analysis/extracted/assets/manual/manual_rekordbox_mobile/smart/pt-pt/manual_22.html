Actuações de DJ

Actuar com a sua unidade XDJ

Modelos aplicáveis: XDJ-AERO/XDJ-RX

Os ficheiros de música podem ser seleccionados no painel do browser e carregados na unidade de DJ.

1Estabeleça a ligação entre o rekordbox (iOS/Android) e a unidade de DJ.

  • Associar a uma unidade de XDJ

Quando a ligação é estabelecida, o painel do browser abre-se.

2Quando toca numa faixa no painel do browser, o painel de ligação (XDJ) é exibido.

As listas de reprodução e listas de tags podem ser utilizadas para organizar ficheiros de uma forma prática ao seleccionar faixas.

  • Organizar música em listas de reprodução

  • Seleccionar música para uma lista de tags

3Quando toca no ícone do deck de uma unidade de DJ, o ficheiro de música é carregado nesse deck.

  • Quando a função AUTOMIX da unidade de DJ (XDJ-AERO) está activada e um ficheiro de música é seleccionado a partir de uma lista de reprodução e carregado, o próximo ficheiro de música também é carregado automaticamente no outro deck.

Nota

O conteúdo do dispositivo USB ligado à unidade de DJ (XDJ-AERO) pode ser explorado e seleccionado tocando em [USB] na barra de ferramentas no painel do browser.

  • Para obter instruções sobre como manusear dispositivos USB, consulte as instruções de funcionamento da respectiva unidade de DJ.

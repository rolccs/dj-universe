Organización de música en listas de reproducción

Las listas de reproducción se pueden crear según sus requerimientos para la actuación de DJ, para organizar la librería.

1Cuando se toca [Prepare] en el menú inicial, la ventana de explorador se abre.

2Toque la categoría 【PLAYLIST】.

Se visualiza una lista de listas de reproducción y carpetas.

3Cuando se toca , la casilla de introducción de nombres de listas de reproducción se abre.

4Introduzca el nombre de la lista de reproducción y luego toque [OK].

La lista de reproducción recién creada se añade a la categoría 【PLAYLIST】.

5Toque la lista de reproducción recién creada para abrirla.

6Toque [Editar] y luego .

Se abre un panel para seleccionar pistas que va a añadir.

7Toque las pistas que quiera añadir a la lista de reproducción. (Se pueden seleccionar múltiples pistas.)

8Toque [OK] para terminar de añadir pistas.

9Toque [OK] para terminar de editar listas de reproducción.

Edición de listas de reproducción existentes

1Toque la categoría 【PLAYLIST】.

Se visualiza una lista de listas de reproducción y carpetas.

2Toque la lista de reproducción que quiere editar para abrirla.

3Toque [Editar] y luego añada o elimine pistas y cambie el orden de las pistas.

Icono

Descripción

Cuando se toca se abre un panel para seleccionar pistas que va a añadir a las listas.

Cuando se arrastra, el orden de las pistas se puede cambiar.

Cuando se toca se visualiza [Quitar]. Toque [Quitar] para eliminar pistas.

Ampliar tabla

4Toque [OK] para terminar de editar listas de reproducción.

Organización de listas de reproducción

1Toque la categoría 【PLAYLIST】.

Se visualiza una lista de listas de reproducción y carpetas.

2Cuando se toca , la casilla de introducción de nombres de carpetas se abre.

3Introduzca el nombre de la carpeta y luego toque [OK].

La carpeta creada se añade a la categoría 【PLAYLIST】.

4Toque [Editar] y luego mueva o elimine la lista de reproducción o la carpeta.

Icono

Descripción

Use esto para crear una lista de reproducción nueva.

Use esto para crear una carpeta nueva.

Las listas de reproducción y las carpetas se pueden mover arrastrándolas. Para mover una lista de reproducción o carpeta al interior de una carpeta de un nivel inferior, arrástrela y póngala sobre la carpeta. Se visualiza el nivel de abajo. Para mover una lista de reproducción o carpeta al interior de una carpeta de un nivel superior, arrástrela al borde izquierdo de la pantalla. Se visualiza el nivel de arriba.

Cuando se toca y mantiene tocado el nombre de la lista de reproducción, la casilla de introducción de nombres de listas de reproducción se abre. Cuando se introduce el nombre de la lista de reproducción y luego se toca [OK], el nombre de la lista de reproducción se edita.

Cuando se toca y mantiene tocado el nombre de la carpeta, la casilla de introducción de nombres de carpetas se abre. Cuando se introduce el nombre de la carpeta y luego se toca [OK], el nombre de la carpeta se edita.

Cuando se toca esto se visualiza [Eliminar]. Toque [Eliminar] para eliminar la lista de reproducción seleccionada.

  • Cuando se elimina una carpeta, todas las listas de reproducción y carpetas de esa carpeta se eliminan también.

Ampliar tabla

5Toque [OK] para terminar de editar las listas de reproducción y las carpetas.

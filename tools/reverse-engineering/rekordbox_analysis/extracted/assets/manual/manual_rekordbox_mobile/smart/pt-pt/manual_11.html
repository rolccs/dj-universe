Ligações

Associar a uma unidade de XDJ

Modelos aplicáveis: XDJ-AERO

Os dispositivos móveis podem ser ligados através de uma LAN sem fios a unidades de DJ da Pioneer DJ que suportam rekordbox (iOS/Android) para trocar dados de música.

1Ligue à rede utilizando a função LAN wireless interna da unidade de DJ.

  • Para obter detalhes sobre os tipos de ligação LAN wireless e o procedimento de ligação, consulte as respectivas instruções de funcionamento da unidade de DJ.

  • XDJ-AERO Guia de ligação

2Ligue o dispositivo móvel no qual o rekordbox (iOS/Android) é instalado à rede LAN wireless.

  • Para obter instruções sobre como criar as ligações LAN wireless, consulte as instruções do dispositivo móvel.

3No menu superior do rekordbox (iOS/Android), toque em [Load].

Quando é encontrada uma unidade de DJ com a qual é possível comunicar, o rekordbox (iOS/Android) tenta ligar-se automaticamente à mesma.

Quando a ligação é estabelecida, o painel do browser abre-se.

Notas

Podem ser ligados até quatro dispositivos em simultâneo à unidade de DJ.

O nome do dispositivo móvel exibido na unidade DJ pode ser verificado e alterado utilizando o rekordbox (iOS/Android).

  • Preferências

Os dispositivos Android definidos para o modo MSC funcionam como dispositivos de armazenamento USB (dispositivos de armazenamento secundário, tais como unidades de memória flash USB) quando ligados a uma unidade de DJ por cabo USB. Nesse caso, os dados não podem ser transferidos utilizando o rekordbox (iOS/Android). Para obter informações ligações entre unidades de DJ e dispositivos de armazenamento USB, consulte as instruções de funcionamento da respectiva unidade de DJ.

Cancelar a ligação com a unidade de DJ

No painel do browser, quando toca em ou [Menu] na barra de ferramentas e, em seguida, regressa ao menu superior, a ligação à unidade de DJ é cancelada.
